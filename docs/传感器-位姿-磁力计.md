# 磁力计


## 1. 模块概览

磁力计用于测量三维磁场矢量，常用于电子罗盘、姿态估计、磁场检测、地磁导航、位置探测。

单独的磁力计可提供绝对航向（磁北），与加速度计/陀螺仪融合后可输出稳定姿态（详见九轴 IMU 文档）。


## 2. 工作原理与适用场景

基于霍尔效应或各向异性磁阻（AMR）/巨磁阻（GMR）/隧道磁阻（TMR）。地球磁场强度约 25~65 µT，磁力计需 µT 量程和高分辨率才能测量地磁矢量。

应用：
- 手机/平板电子罗盘；
- 无人机航向辅助；
- AR/VR 头部方向；
- 室内磁场定位（iBeacon 类）；
- 门窗开关检测（与磁铁配合）；
- 电机电流检测（间接）。

## 5. 硬件连接

- 远离干扰源：电机、电池、扬声器、铁磁性材料（含螺丝、屏蔽罩）；
- 模块放置：尽量远离主 PCB 大电流走线（DC-DC 开关电流会耦合磁场）；
- I2C 上拉：4.7kΩ；
- 多数芯片工作电压 1.8V / 3.3V；
- 模块层叠：部分九轴模块将磁力计放到独立小板上，避免主 PCB 干扰。


## 6. 通信方式

- I2C：标准 100/400 kHz；
  - HMC5883L 默认地址 0x1E；
  - QMC5883L 默认地址 0x0D；
  - LIS3MDL 默认 0x1C（SDO=高）或 0x1E（SDO=低）；
  - AK09911 默认 0x0C；
  - AK09918 默认 0x0C；
- SPI：部分型号支持；
- 单轴模拟：直接 ADC 读取。


## 7. 初始化流程

1. 验证 WHO_AM_I；
2. 软复位；
3. 设置量程、ODR、带宽；
4. 设置测量模式（单次/连续）；
5. 触发测量（如 HMC5883L 需写 0x01 到 Mode 寄存器）；
6. 读取 X/Y/Z 数据。

## 9. 数据格式与单位

- 单位：µT（特斯拉）或 mGauss（毫高斯），1 µT = 10 mGauss；
- 字节序：MSB 在前；
- 数据更新：单次模式下需主动触发，连续模式下按 ODR 自动更新；
- 量程与灵敏度对应：高斯 → µT 转换 1 G = 100 µT；
- 地磁参考：水平分量约 20~50 µT（不同纬度不同）；
- 磁偏角：磁北与真北夹角（一般 0~10°，需查当地磁偏角表）。


## 10. 校准与滤波

### 10.1 硬铁校准

磁力计周围恒定铁磁材料造成固定偏移。绕三轴旋转采集 N 个样本，椭球中心即硬铁偏移：

```python
bx = (max_x + min_x) / 2
by = (max_y + min_y) / 2
bz = (max_z + min_z) / 2
```

### 10.2 软铁校准

非均匀磁场导致椭球畸变，需构造 3×3 校正矩阵把椭球拉成球体。

### 10.3 简化的 9 点 / 椭球拟合

旋转采集数百点 → 拟合椭球 → 求中心和变换矩阵。

```python
# 椭球拟合示例伪代码
import numpy as np
```

## 11. 错误处理

- **磁干扰**：电机、电池、电源模块、磁铁、扬声器；
- **软启动电流冲击**：DC-DC 启动磁场尖峰；
- **温度漂移**：低温/高温偏移；
- **磁异常**：靠近磁铁时饱和；
- **自校准失败**：用户未旋转充分。



- WHO_AM_I 错：检查 I2C 地址（注意 HMC/QMC 不兼容）；
- 数据全 0：未触发测量；
- 数据饱和：磁场过强，靠近磁铁；
- 总线卡死：I2C 恢复。


## 12. 低功耗

- 单次测量模式：测量后自动进入低功耗；
- 测量间隔可低至几 ms；
- 多数芯片 1 µA 以下待机电流。

```python
class Magnetometer:
    def __init__(self, bus, address=0x0D):  # QMC5883L 默认
        self.bus = bus
        self.address = address
        self.bias = (0.0, 0.0, 0.0)

    def begin(self):
        pass

    def read_mag(self):
        """返回 mx, my, mz, 单位 µT。"""
        pass

    def calibrate(self, samples=1000):
        """绕三轴旋转采集，返回硬铁偏移。"""
        pass

    def get_corrected_mag(self):
        mx, my, mz = self.read_mag()
        bx, by, bz = self.bias
        return mx - bx, my - by, mz - bz

    def heading(self, mx, my):
        """水平航向角 = atan2(my, mx) * 180/π，加磁偏角修正。"""
        import math
        return math.degrees(math.atan2(my, mx))


mag = Magnetometer(i2c, address=0x0D)
mag.begin()
mag.calibrate(samples=1000)
while True:
    mx, my, mz = mag.get_corrected_mag()
    deg = mag.heading(mx, my)
    print("mag=(%.1f,%.1f,%.1f) heading=%.1f°" % (mx, my, mz, deg))
```


## 14. 调试方法

1. 静止读数应当稳定；
2. 旋转 90°/180°/270° 观察航向变化；
3. 水平面旋转 → 航向连续变化；
4. 翻转模块 → Z 轴读数变化；
5. 远离电脑/手机/扬声器测试磁场干扰。


## 15. 常见问题

- **HMC5883L 读不到数据**：必须先写 Mode 寄存器触发单次测量；
- **QMC5883L 地址错**：默认 0x0D 而非 0x1E，与 HMC5883L 不兼容；
- **磁航向偏差**：磁偏角未补偿；
- **姿态指向偏差**：模块不在水平面或干扰大；
- **数据溢出**：磁场过强。
