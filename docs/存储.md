# 存储总览

嵌入式持久化存储。DriverKit 收录 **3 个存储模块**，覆盖小参数到大容量数据。

## 模块速查

| 模块 | 说明 | 文档 |
|---|---|---|
| EEPROM | 按字节读写，小参数（1 Kbit ~ 1 Mbit） | [打开](./存储-非易失-EEPROM) |
| Flash | NOR 大容量，文件 / 字库（1 Mbit ~ 1 Gbit+） | [打开](./存储-非易失-Flash) |
| SD 卡 | SDHC/SDXC，便携大容量（2 GB ~ 1 TB+） | [打开](./存储-外部-SD卡) |

## 选型速查表

| 应用场景 | 推荐 |
|---|---|
| 设备参数（KB 级） | [EEPROM](./存储-非易失-EEPROM) AT24C 系列 |
| 频繁写日志 | [EEPROM](./存储-非易失-EEPROM) FRAM FM24CL 系列 |
| 中文字库（MB 级） | [Flash](./存储-非易失-Flash) W25Q 系列 |
| 文件系统（MB 级） | [Flash](./存储-非易失-Flash) + LittleFS |
| 数据记录（GB 级） | [SD 卡](./存储-外部-SD卡) |
| 视频监控 | [SD 卡](./存储-外部-SD卡) 高速（U3/V30） |
| 树莓派系统盘 | [SD 卡](./存储-外部-SD卡) A2 + 32 GB+ |

## 共性知识

- **EEPROM** 按字节写，写次数 10⁶（FRAM 无限制）
- **Flash** 按页 / 扇区操作，先擦后写，写次数 10⁵
- **SD 卡** 文件系统兼容 PC，但突然掉电不安全
- **掉电保护**：CRC 校验、双份备份
- **磨损均衡**：LittleFS 自动管理，FAT 需应用层处理
