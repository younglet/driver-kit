# 旋转编码器


## 1. 模块概览

旋转编码器（Rotary Encoder）是一种测量旋转角度或角速度的输入设备。常见于：
- 音量旋钮；
- 工业控制旋钮；
- 数控机床手动轮（手轮）；
- 电机测速；
- 机器人关节；
- 雷达天线；
- 相机对焦环；
- 平衡车/独轮车（脚踏位置）。

类型：
- **增量式编码器**：AB 相输出，根据脉冲数计算角度和方向；
- **绝对值编码器**：每个位置唯一编码（见"编码器"文档）；
- **机械式旋转编码器**：旋钮型，EC11/EC12/EC16 等；
- **光学式**：高分辨率；
- **磁性式**：抗污染；
- **电容式**：高精度无接触。


## 2. 工作原理与适用场景

### 2.1 增量式

- 两个开关 A、B，按特定相位错开；
- 旋转时 A、B 输出方波，相位差 90°（正交）；
- 顺时针：A 先于 B；
- 逆时针：B 先于 A；
- 计数：A 上升沿时检查 B 的电平 → 方向；
- 4 倍频：A、B 的上升沿和下降沿都计数（分辨率 ×4）。

### 2.2 CPR / PPR

- **CPR**：每转计数（cycles per revolution）；
- **PPR**：每转脉冲（pulses per revolution）；
- 4 倍频后：分辨率 = CPR × 4；
- 例如 EC11 = 20 CPR，4 倍频 = 80 脉冲/转。

### 2.3 速度测量

- **M 法（测频）**：固定时间窗内数脉冲，适合高速；
- **T 法（测周）**：测量相邻脉冲间隔，适合低速；
- **M/T 法**：结合两者，精度高。

## 5. 硬件连接

### 5.1 基本电路

- A、B 两根信号线接 MCU；
- 内置上拉电阻（多数 EC11/EC12）；
- 按键（SW）：可选；
- 公共端 COM：接 GND 或 VCC（取决于电路）；
- 串联电阻：100Ω 限流；
- TVS 防静电。

### 5.2 长线传输

- 加施密特触发器（74HC14）；
- 差分传输（高速 / 长距离）；
- 双绞线减少干扰。

### 5.3 抗干扰

- 滤波电容（100nF）；
- 屏蔽线；
- 软件滤波。


## 6. 通信方式

- AB 相（直接 GPIO）；
- I2C：AS5600 等磁性编码器；
- SPI：AS5048A 等；
- SSI：绝对值编码器；
- 模拟量：少数简单编码器。


## 7. 初始化流程

1. 配置 GPIO（A、B 输入 + 上下拉）；
2. 配置中断（A/B 边沿触发）；
3. 计数器清零；
4. 启动。


## 8. 驱动接口

```python
class RotaryEncoder:
    """增量式旋转编码器。"""
    def __init__(self, pin_a, pin_b, pin_sw=None):
        self.a = pin_a
        self.b = pin_b
        self.sw = pin_sw
        self.count = 0
        self.last_a = False
        self.last_dir = 0  # -1, 0, +1

    def begin(self):
        """初始化 GPIO 和中断。"""
        pass

    def update(self):
        """由中断或定时器调用。"""
        a = self.a.value()
        b = self.b.value()
        # 状态机解码
        if a != self.last_a:
            if a == b:
                self.count += 1
                self.last_dir = 1
            else:
                self.count -= 1
                self.last_dir = -1
        self.last_a = a

    def read_count(self):
        return self.count

    def reset(self):
        self.count = 0

    def is_pressed(self):
        return self.sw and self.sw.value() == 0


class SpeedEncoder:
    """测速编码器（M 法）。"""
    def __init__(self, encoder, cpr=20):
        self.encoder = encoder
        self.cpr = cpr

    def speed_rpm(self, dt_ms):
        """返回 rpm。"""
        delta = self.encoder.read_count()
        return (delta * 60 * 1000) / (self.cpr * dt_ms)
```



### 12.1 M 法（测频）

```python
def m_method(count_delta, dt_seconds, cpr):
    """固定时间窗内的脉冲数。"""
    return count_delta / (dt_seconds * cpr) * 60  # rpm
```

### 12.2 T 法（测周）

测量相邻脉冲时间：

```python
def t_method(pulse_period, cpr):
    """每脉冲时间。"""
    return 60 / (pulse_period * cpr)  # rpm
```

### 12.3 M/T 法

M 法 + T 法结合，宽速度范围。


## 9. 数据格式与单位

- 增量：脉冲数（带符号表示方向）；
- 速度：脉冲数 / 时间 = 频率 → rpm；
- 角度：脉冲数 / CPR × 360°。


## 10. 校准与滤波

- 中值滤波：去除脉冲噪声；
- 滑动平均：降低速度抖动；
- 卡尔曼滤波：状态估计（角度 + 速度）。


## 11. 中断与 DMA

- 边沿中断：A/B 任意边沿触发（4 倍频）；
- 定时器中断：定时计算速度；
- DMA：高速计数。

## 12. 错误处理

- **抖动**：状态机过滤（必须）；
- **误计数**：长线干扰，加滤波；
- **漏计数**：中断处理时间过长；
- **A/B 反接**：方向反，软件或硬件交换；
- **按键抖动**：去抖。


## 13. 低功耗

- 关闭未使用的定时器；
- 按键中断唤醒（编码器可作为唤醒源）；
- 长时间不操作进入休眠。


## 14. 示例代码

```python

enc.begin()
while True:

```


## 15. 调试方法

1. 手动旋转测计数变化；
2. 顺时针 +1，逆时针 -1；
3. 用示波器看 A、B 波形；
4. 验证 4 倍频（每转计数 × 4 = 80）；
5. 测试按键按下/松开。


## 16. 常见问题

- **方向反**：A、B 接线反或软件 reverse；
- **不计数**：GPIO 配置错，未启用中断；
- **计数跳变**：抖动未消除；
- **按键不响应**：按键接错（多数带按键）、GPIO 模式错；
- **高速漏计**：中断处理太慢或 MCU 频率不够。


## 17. 参考资料
