## 文件命名规则

每一种硬件只使用一个文件，格式为：

```text
分类-领域-模块.md
```

示例：

```text
传感器-位姿-三轴陀螺仪.md
通信-总线-I2C.md
执行器-运动-直流电机.md
```

新增模块后，会同时更新本页和模块列表。
