# 无刷电机（BLDC）


## 1. 模块概览

无刷电机（Brushless DC, BLDC）通过电子换向实现转子持续旋转，无机械磨损，寿命长、效率高、噪音低。常见于：
- 无人机螺旋桨；
- 电动工具（电钻、风扇、吹风机）；
- 平衡车、独轮车；
- 电动滑板；
- 硬盘主轴；
- 风扇、泵；
- 工业机器人关节。


## 2. 工作原理与适用场景

定子线圈通电产生旋转磁场，吸引/排斥永磁转子。电子调速器（ESC）按转子位置切换三相绕组通电顺序。

驱动方式：
- **方波驱动（6-step）**：每 60° 切换一次相序；
- **正弦波驱动**：连续正弦电流，转矩波动小；
- **FOC（磁场定向控制）**：矢量控制，最平滑、最高效。

转子位置检测：
- **有感（带霍尔/编码器）**：3 个霍尔传感器或编码器；
- **无感（反电动势法）**：通过检测反向 EMF 估算转子位置。

## 5. 硬件连接

- 三相线（A/B/C）：粗线（如 14~16 AWG）；
- 霍尔传感器线：5 线（VCC/GND/A/B/C）；
- 高电流路径：电源正负极粗线、低内阻连接器（XT60、XT90、Amass）；
- 散热：长时间大电流需散热片/风扇；
- 静电保护：MOSFET 栅极保护；
- 电调安装：远离高频干扰（磁环、屏蔽）；
- **不要在无负载下长时间通电测试**，可能损坏电机；
- 电调校准：每次换电机或电调都要做油门行程校准。


## 6. 通信方式

- **PWM**：50 Hz 50 Hz（500~2500 µs）；
- **Oneshot125**：单脉冲 125~250 µs（8 kHz 协议）；
- **Multishot**：5~25 µs（高速）；
- **DShot150/300/600/1200**：数字协议（带校验）；
- **ProShot/ProfiLED**：进阶协议；
- **CAN**：高端电调；
- **UART**：配置和遥测。


## 7. 初始化流程

1. 接线（电源、电机三相、信号）；
2. 电调校准油门行程（首次或更换时）；
3. 写入 PWM 最低值 = 最低转速；
4. 设置方向（必要时交换任意两相）；
5. 配置 PID（电调内）；
6. 启动。

### 10.1 PWM（标准）

50 Hz，1 ms = 最低，2 ms = 最高。简单但响应慢。

### 10.2 Oneshot125

8 kHz 协议，125~250 µs。低延迟。

### 10.3 DShot

数字协议（带校验）：
- DShot150：150 kbaud；
- DShot300：300 kbaud；
- DShot600：600 kbaud；
- DShot1200：1.2 Mbaud。

需要 MCU 硬件支持（DMA + 定时器）。

### 10.4 BLHeli_32 / AM32

开源 BLDC 电调固件，支持 DShot、BiDir 等。



矢量控制算法，电流分解为 d/q 轴独立控制：
- 转矩波动极小；
- 效率最高（90%+）；
- 需要较多计算（DSP/FPGA/高速 MCU）；
- SimpleFOC 开源库支持 Arduino、STM32、ESP32。


## 9. 数据格式与单位

- PWM：脉宽 1000~2000 µs（典型）；
- DShot：11-bit 数字值 + 4-bit CRC；
- RPM：电调遥测（电子转速）；
- 电流/电压：遥测。


## 10. 低功耗

- 反电动势制动回收（BLDC regen）；
- 主动短路制动；
- 闲置时关闭驱动。

## 12. 调试方法

1. 检查三相线接线（A/B/C 对应）；
2. 听启动声音（是否正常转动）；
3. 测电流（空载 1~2A，负载增大）；
4. 测温度（长时间运行 < 80°C）；
5. 调 PID（电调内）；
6. DShot 配置确认双向遥测。


## 13. 常见问题

- **电机抖动不转**：相序错、霍尔接线错、电调未校准；
- **启动困难**：无感启动失败、负载过大；
- **电调烧毁**：PWM 频率不对、散热不足、堵转过流；
- **电机过热**：长时间大电流、风冷不足；
- **噪音大**：方波驱动、轴承磨损；
- **振动大**：动平衡差；
- **DShot 无响应**：接线错、协议不匹配。


## 14. 参考资料

- **安全**：螺旋桨等高速旋转设备极其危险，测试时远离人；

## 15. 反电动势法（BEMF 无感）

通过检测未通电相的反向 EMF 估算转子位置：
- 0 速度无法启动，需要开环启动；
- 中高速稳定；
- 启动算法复杂。
