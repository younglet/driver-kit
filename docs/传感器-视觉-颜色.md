# 颜色传感器


## 1. 模块概览

颜色传感器通过分光或光电二极管检测 RGB 或更精细光谱分量，用于：
- 颜色识别与匹配；
- LED 分选；
- 工业颜色检测；
- 自动色温测量；
- 灯光颜色校准；
- 印刷/纺织检测；
- 教育/创客项目。

类型：
- **RGB 颜色传感器**：TCS34725、APDS-9960、TCS3200；
- **光谱传感器**：AS7341（11 通道）、AS7265x；
- **色温传感器**：TCS3430；
- **XYZ 三刺激值**：高端工业；
- **近红外/多光谱**：农业、食品检测。


## 2. 工作原理与适用场景

### 2.1 RGB 颜色

光电二极管 + RGB 滤光片阵列，分别测量红/绿/蓝光强度。三刺激值 X/Y/Z 可由 RGB 推算。

### 2.2 多通道光谱

多个独立光电二极管 + 不同波长滤光片（8~18 通道），输出每个波段强度，适合精细测量。

### 2.3 白平衡/色温

通过 RGBC（红、绿、蓝、Clear）四通道计算色温和照度。

**选型建议**：
- 入门：TCS3200（频率输出，简单）；
- RGB 颜色识别：TCS34725、APDS-9960；
- 色温/白平衡：TCS3430、BH1749；
- 多光谱（专业）：AS7341、AS7265x；
- 接近 + 手势：APDS-9960；
- UV 检测：VEML6075。


## 5. 硬件连接

- I2C 上拉：4.7kΩ；
- INT：中断输出（阈值）；
- LED 引脚：TCS34725 等内置 LED 驱动器；
- 透镜：光电二极管需透镜聚焦；
- 红外滤除：TCS34725 内置红外阻挡滤光片；
- 接近测量：APDS-9960 红外 LED + 检测器；
- 注意光照强度：强光下需降低增益或缩短积分时间。


## 6. 通信方式

- I2C：标准 100/400 kHz；
- 频率输出：TCS3200（输出方波频率与颜色分量成正比）；
- 部分支持 SMBus Alert。


## 7. 初始化流程

1. 配置 I2C；
2. 读 CHIP_ID；
3. 软复位；
4. 设置积分时间；
5. 设置增益（1x/4x/16x/60x）；
6. 启动测量；
7. 等待中断或轮询；
8. 读取 RGB/Clear 数据。

## 9. 数据格式与单位

- RGB：R/G/B 三通道 16-bit 计数；
- Clear：白光强度；
- Lux：照度；
- CCT：色温（K）；
- CIE 1931 (x, y)：色度坐标；
- HSI：色相（Hue）/饱和度（Saturation）/亮度（Intensity）。


## 10. 校准与滤波

- 白点校准：在已知色温光源下读取 RGB，确定白点偏移；
- 矩阵校准：使用色卡（如 X-Rite ColorChecker）进行多色校准；
- 暗补偿：黑平衡校正；
- 距离归一化：传感器与目标距离不同时，需考虑散射衰减。


## 11. 中断与 DMA

- 阈值中断：Clear 通道超过阈值；
- 部分芯片支持 FIFO（多次采样缓存）。


## 12. 错误处理

- I2C 错误：检查地址、上拉；
- 读数饱和：调整增益或积分时间；
- 颜色判定错误：白平衡未校准；
- 干扰光：环境光变化大时，需周期性重新校准。


## 13. 低功耗

- 单次测量后休眠；
- 待机 < 1µA；
- 间歇测量：1 Hz 足够应用。

## 15. 调试方法

1. 测试不同颜色物体（红、绿、蓝、白、黑）；
2. 测试不同光源（日光、白炽灯、LED）；
3. 测试色温估算；
4. 对照色卡验证精度；
5. 测试环境光干扰。


## 16. 常见问题

- **颜色判定不准**：白平衡未校准；
- **饱和**：强光，调整增益；
- **TCS3200 频率法误差大**：使用集成 IC（TCS34725）；
- **接近功能不响应**：APDS-9960 LED 未启用；
- **红外干扰**：使用内置 IR 阻挡滤光片型号。

## 18. RGB → HSI/HSV 转换

```python
def rgb_to_hsv(r, g, b):
    r, g, b = r/255.0, g/255.0, b/255.0
    mx, mn = max(r, g, b), min(r, g, b)
    df = mx - mn
    if mx == r:
        h = ((g - b) / df) % 6
    elif mx == g:
        h = (b - r) / df + 2
    else:
        h = (r - g) / df + 4
    h *= 60
    s = 0 if mx == 0 else df / mx
    v = mx
    return h, s, v
```
