# 水泵


## 1. 模块概览

水泵（Water Pump）通过电机驱动输送液体。常见类型：
- **直流隔膜泵**：低流量、高压力；
- **离心泵**：高流量、低压力；
- **蠕动泵（Peristaltic）**：精确计量、无污染；
- **齿轮泵**：精确流量、高压；
- **活塞泵**：高压脉动；
- **潜水泵**：水下作业；
- **微型真空泵**：抽气、抽水。

应用：
- 水冷系统；
- 自动灌溉；
- 喷泉、景观；
- 实验仪器；
- 医疗（透析、输液）；
- 化学液输送；
- 咖啡机；
- 3D 打印机（冷却液循环）；
- 汽车冷却。


## 2. 工作原理与适用场景

### 2.1 隔膜泵

电机驱动偏心轮 → 隔膜往复运动 → 进出水单向阀交替开闭 → 液体单向流动。

### 2.2 离心泵

电机驱动叶轮高速旋转 → 液体在离心力作用下甩出 → 形成水流。

### 2.3 蠕动泵

电机驱动滚轮 → 挤压软管 → 推动液体单向流动，液体只接触软管内壁。

### 2.4 齿轮泵

两个齿轮啮合旋转 → 液体在齿槽间被推送。

## 5. 硬件连接

- **驱动电流**：12V 隔膜泵 1~3A，需 MOSFET 或专用驱动；
- **续流二极管**：电机线圈反并；
- **去耦电容**：100nF + 100µF；
- **PWM 调速**：部分泵支持（频率 1~10 kHz）；
- **H 桥方向控制**：蠕动泵可正反转（吸/排）；
- **进水管**：避免弯曲、漏气；
- **出水管**：注意背压；
- **防水**：潜水泵 IP68；
- **散热**：大功率泵长时间运行需散热；
- **过滤**：进水口加过滤网。


## 6. 通信方式

- **GPIO/MOSFET**：基本开关控制；
- **PWM**：调速（多数泵支持）；
- **H 桥 + PWM**：蠕动泵正反转控制；
- **UART/RS485**：智能泵控制（如 Kamoer KPP）；
- **模拟电压**：部分工业泵。


## 7. 初始化流程

1. 接线（电源、控制信号）；
2. 检查进水管路密封；
3. 灌注液体（首次使用前）；
4. 测试运行（低速 → 高速）；
5. 校准流量（蠕动泵）。


## 8. 驱动接口

```python
class Pump:
    def __init__(self, pin):
        self.pin = pin

    def begin(self):
        pass

    def on(self):
        self.pin.high()

    def off(self):
        self.pin.low()


class PwmPump:
    """可调速泵。"""
    def __init__(self, pwm):
        self.pwm = pwm

    def begin(self):
        self.pwm.duty(0)

    def set_speed(self, percent):
        """0~100。"""
        self.pwm.duty(percent)


class PeristalticPump:
    """蠕动泵（双向）。"""
    def __init__(self, in1, in2, pwm):
        self.in1 = in1
        self.in2 = in2
        self.pwm = pwm

    def begin(self):
        self.stop()

    def forward(self, speed):
        self.in1.high()
        self.in2.low()
        self.pwm.duty(speed)

    def backward(self, speed):
        self.in1.low()
        self.in2.high()
        self.pwm.duty(speed)

    def stop(self):
        self.in1.low()
        self.in2.low()
        self.pwm.duty(0)

    def dispense_ml(self, ml, rate_ml_min=10):
        """按流量和时间定量输送。"""
        seconds = ml * 60 / rate_ml_min
        self.forward(50)
        time.sleep(seconds)
        self.stop()
```


## 9. 数据格式与单位

- PWM 占空比：0~100%；
- 流量：L/min、mL/min；
- 压力：MPa、bar、kPa；
- 定量输送：mL、mL/min；
- 状态：电压、电流、温度。


## 10. 校准与滤波

- 蠕动泵：测量实际流量校准（ml/pulse 或 ml/s）；
- 隔膜泵：流量-转速特性曲线；
- 离心泵：扬程-流量曲线（H-Q）；
- 储存校准数据版本号 + CRC。


## 11. 错误处理

- **不吸水**：进水口漏气、自吸能力不够；
- **流量不足**：泵内堵塞、背压过高；
- **过热**：长时间大负载、堵塞；
- **噪声大**：气蚀、堵塞；
- **电机烧毁**：堵转、过载；
- **泄漏**：密封件老化。



- 高压液体危险；
- 化学介质需耐腐蚀泵；
- 食品/医疗需食品级认证；
- 户外使用注意防水防雷。


## 12. 低功耗

- 间歇运行；
- PWM 调速（流量需求不高时降速）；
- 待机时关闭电源。

## 14. 调试方法

1. 测试空载运行；
2. 测试带载流量；
3. 测扬程（静扬程 vs 实际扬程）；
4. 测电流（负载变化）；
5. 测温度（长时间运行）；
6. 检查密封（无气泡、无泄漏）。


## 15. 常见问题

- **不吸水**：进水口漏气或缺水；
- **流量不足**：滤网堵塞、电机功率不足；
- **噪声大**：气蚀（进水不足）；
- **泵过热**：长时间大负载；
- **电机烧毁**：堵转时间过长；
- **蠕动泵精度差**：管路老化、流量校准失效。
