# 电磁阀


## 1. 模块概览

电磁阀（Solenoid Valve）是用电磁铁控制的流体阀，常用于：
- 气动控制系统（机械手、气缸）；
- 液压控制；
- 水流控制（洗衣机、洗碗机、净水器）；
- 燃油控制；
- 医疗设备（呼吸机、麻醉机）；
- 自动灌溉；
- 咖啡机；
- 工业自动化。

类型：
- **直动式**：电磁力直接启闭阀芯；
- **先导式**：电磁力启闭先导孔，利用压差启闭主阀；
- **比例阀**：连续控制流量/压力；
- **二位二通（2/2）**：单通路；
- **二位三通（3/2）**：单通路 + 排气；
- **二位五通（5/2）**：双通路 + 排气；
- **气动阀、油压阀、水阀**。


## 2. 工作原理与适用场景

线圈通电产生磁场，推动阀芯移动：
- 直动式：电磁力 > 流体压力 → 直接打开；
- 先导式：电磁力打开先导孔，主阀靠压差打开；
- 失电复位：弹簧复位（常闭）或保持（常开）。

## 5. 硬件连接

- **驱动电流**：DC 24V 阀通常 200~500 mA，需三极管/MOSFET 驱动；
- **续流二极管**：反并 1N4007，防止反向 EMF；
- **光耦隔离**：工业应用建议隔离；
- **手动复位**：部分阀有手动开关；
- **电磁干扰**：感性负载加 RC 吸收；
- **密封**：进水阀注意 IP 等级；
- **温度**：高温环境注意阀体材质；
- **防爆**：燃气等场合需防爆认证。


## 6. 通信方式

- **GPIO**：直接控制（多数）；
- **PWM**：比例阀控制流量/压力；
- **模拟电压**：±10V 或 0~10V；
- **电流信号**：4~20 mA 工业标准；
- **现场总线**：CANopen、PROFIBUS、DeviceNet；
- **IO-Link**：现代工业总线。


## 7. 初始化流程

1. 配置 GPIO/MOSFET 驱动；
2. 设置初始为关闭；
3. 上电稳定后开始控制；
4. 比例阀需配置控制参数（PWM/电压映射）。


## 8. 驱动接口

```python
class SolenoidValve:
    def __init__(self, pin, active_high=True):
        self.pin = pin
        self.active_high = active_high

    def begin(self):
        self.close()

    def open(self):
        if self.active_high:
            self.pin.high()
        else:
            self.pin.low()

    def close(self):
        if self.active_high:
            self.pin.low()
        else:
            self.pin.high()


class ProportionalValve:
    """比例阀。"""
    def __init__(self, pwm):
        self.pwm = pwm

    def begin(self):
        pass

    def set_flow(self, percent):
        """0~100。"""
        self.pwm.duty(percent)


class LatchingSolenoid:
    """脉冲型电磁阀（节能）。"""
    def __init__(self, open_pin, close_pin):
        self.open = open_pin
        self.close = close_pin

    def open(self):
        self.open.high()
        time.sleep_ms(50)
        self.open.low()

    def close(self):
        self.close.high()
        time.sleep_ms(50)
        self.close.low()
```


## 9. 数据格式与单位

- 开/关：布尔；
- 比例阀：占空比或电压；
- 模拟输入：4~20 mA 或 0~10V；
- 总线：寄存器或 PDO（过程数据对象）。


## 10. 错误处理

- **阀门卡死**：杂质、结垢、生锈；
- **泄漏**：密封件老化；
- **不响应**：线圈断、驱动故障；
- **流量下降**：阀芯磨损、过滤网堵塞；
- **响应慢**：先导孔堵塞、压差不足；
- **寿命终结**：触点氧化、弹簧疲劳。



- 高压气体/液体危险；
- 燃气阀泄漏可能爆炸，需定期检查；
- 防爆场合必须用防爆认证阀；
- 医疗设备用医用级阀。



- **不响应**：检查电源、驱动电路；
- **泄漏**：更换密封件；
- **流量不足**：检查阀芯、压力差；
- **响应慢**：检查先导孔、过滤网；
- **总线通信失败**：检查地址、波特率、终端电阻。


## 11. 低功耗

- 保持型阀：脉冲触发、保持；
- 占空比控制：间歇开启节能；
- 比例阀：根据需要调节。

## 13. 调试方法

1. 听吸合声（咔嗒）；
2. 测线圈电压；
3. 测阀后压力/流量；
4. 检查密封（无泄漏）；
5. 长时间运行温度检查；
6. 比例阀用万用表测控制电压。


## 14. 常见问题

- **不响应**：电源未接、驱动未工作；
- **响应慢**：先导孔堵塞；
- **泄漏**：密封件老化、阀座磨损；
- **流量不足**：阀芯磨损、压力差不足；
- **过热**：长时间通电、散热不足；
- **噪音大**：交流驱动加静音电路。
