# 超声波测距传感器


## 1. 模块概览

超声波测距传感器通过发射超声波脉冲，接收反射波，按时间差计算距离。常用于：
- 机器人避障；
- 倒车雷达；
- 液位测量；
- 距离报警；
- 智能小车；
- 工业测距。

优点：不受颜色/光线影响；缺点：声波锥角大、难以检测细小目标、泡沫/织物吸收严重。


## 2. 工作原理与适用场景

声速约 343 m/s（20°C 空气）。测距公式：

```text
距离 = (回波时间 × 声速) / 2
```

由于空气温度影响声速，精确测距需要温度补偿：

```text
声速 = 331.3 + 0.606 × T (m/s)
T 是摄氏度
```

## 5. 硬件连接

- HC-SR04：VCC（5V）、GND、Trig（输入）、Echo（输出）；
- 注意：HC-SR04 的 Echo 输出 5V，与 3.3V MCU 连接需电平转换（电阻分压或 BSS138 MOSFET）；
- JSN-SR04T：3.3V/5V 兼容；
- 防水探头需要独立线缆（5m/10m 等规格）；
- 远离噪声源（电机、风扇、蜂鸣器）；
- 软硬件滤波：去除毛刺（多次测量取中值）。


## 6. 通信方式

- Trig/Echo：发出 10µs 高脉冲触发，Echo 输出与距离成正比的高电平；
- UART：9600~115200 baud，发送 ASCII 指令（如 0x55），返回距离；
- I2C：MB 系列、KS103 等。


## 7. 初始化流程

1. 配置 GPIO（Trig 输出、Echo 输入）；
2. 配置定时器捕获 Echo 脉宽；
3. 配置 UART/I2C（如使用）；
4. 启动循环测量；
5. 温度补偿（可选）。

## 9. 数据格式与单位

- 单位：cm 或 mm；
- HC-SR04 Echo 高电平脉宽 = 距离 cm × 58 µs（往返，cm/µs）；
  - 实际：距离 cm = 时间 µs / 58；
- MB 系列 UART 输出 ASCII 字符串（如 "R123\r\n"）；
- I2C 输出 16-bit 值（cm 或 mm）。

```python
def sound_speed(t_celsius):
    return 331.3 + 0.606 * t_celsius  # m/s

def distance_us(time_us, t=20):
    c = sound_speed(t) * 1e-6 * 1e2  # cm/µs
    return time_us * c / 2
```


## 11. 中断与 DMA

- 使用定时器捕获 Echo 上升沿和下降沿；
- 单次测量时间约 30 ms（远距 5m 时）；
- 测量间隔应 > 60 ms 避免回波重叠。


## 12. 错误处理

- Echo 超时（>30ms）：目标太远或吸收；
- 距离过小（< 2 cm）：盲区；
- 距离跳变：目标物体边缘或多个反射路径；
- 测量失败：传感器未连接、电源不稳。


## 13. 低功耗

- 仅测量时上电；
- 静态功耗 < 2 mA；
- 可用 MOSFET 控制 VCC。


## 14. 示例代码

```python
class Ultrasonic:
    def __init__(self, trig, echo):
        self.trig = trig
        self.echo = echo

    def measure_cm(self, timeout_ms=30):
        """单次测量，超时返回 -1。"""
        self.trig.low()
        time.sleep_us(2)
        self.trig.high()
        time.sleep_us(10)
        self.trig.low()

        # 等待 Echo 高电平开始
        if not wait_high(self.echo, timeout_ms):
            return -1
        t_start = time.ticks_us()
        if not wait_low(self.echo, timeout_ms):
            return -1
        t_end = time.ticks_us()
        us = t_end - t_start
        return us / 58.0

    def measure_filtered(self, n=5):
        samples = []
        for _ in range(n):
            d = self.measure_cm()
            if d > 0:
                samples.append(d)
        if not samples:
            return -1
        return statistics.median(samples)


us = Ultrasonic(trig_pin, echo_pin)
while True:
    cm = us.measure_filtered()
    print("distance: %.1f cm" % cm)
    time.sleep_ms(100)
```


## 15. 调试方法

1. 测试 1m 内壁反射距离；
2. 对着远处墙面观察稳定性；
3. 温度变化时观察声速影响；
4. 检查 Trig 波形（10µs 脉冲）；
5. 检查 Echo 脉宽与距离对应关系。


## 16. 常见问题

- **HC-SR04 一直返回 0**：电源不稳或 Trig/Echo 接错；
- **Echo 5V 烧毁 MCU**：必须加电平转换；
- **测距精度差**：温度未补偿、目标表面太斜；
- **液位不准**：容器口太小导致多次反射；
- **避障距离抖动**：使用中值滤波；
- **HC-SR04 离近 0 距离**：盲区，建议 > 2cm。
