# 时钟总览

实时时钟（RTC，Real-Time Clock）是嵌入式系统的时间基准。DriverKit 收录 **1 个时钟模块**。

## 模块速查

| 模块 | 说明 | 文档 |
|---|---|---|
| RTC | 32.768 kHz + 备用电池 | [打开](./时钟-实时时钟-RTC) |

## 选型速查表

| 应用场景 | 推荐 |
|---|---|
| 入门 / 创客 | DS1307（依赖外接晶振，精度 ±2 ppm） |
| 高精度 / 万年历 | DS3231（内置温度补偿晶振，±2 ppm ≈ ±1 min/年） |
| 超低功耗 | PCF8563 / RX8900 |
| 国产替代 | SD2405 / SD3088 |
| 网络时间同步 | NTP（Wi-Fi / 以太网） |
| 卫星授时 | GNSS PPS |

## 共性知识

- **CR2032 备用电池** 提供断电保持（5~10 年）
- **32.768 kHz 晶振** 是时基
- **BCD 格式** 存储时间（0x25 = 25 秒）
- **闰年补偿** 多数 RTC 自动处理
- **报警中断** 可唤醒 MCU
