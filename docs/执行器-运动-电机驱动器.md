# 电机驱动器


## 1. 模块概览

电机驱动器（Motor Driver）是连接 MCU 与电机的功率电子器件，将 MCU 的低功率控制信号转换为电机驱动所需的高功率信号。常见类型：

- **H 桥驱动器**：用于直流电机（BDC）；
- **三相驱动器**：用于无刷电机（BLDC）、永磁同步（PMSM）；
- **步进电机驱动器**：用于步进电机（详见步进电机文档）；
- **伺服驱动器**：用于伺服电机（详见伺服电机文档）。

本文重点为通用 H 桥和三相 BLDC/PMSM 驱动器。


## 2. 工作原理与适用场景

### 2.1 H 桥

4 个开关（MOSFET/IGBT）组成 H 形，电机在中间。通过控制开关组合：
- 正转：上左 + 下右；
- 反转：上右 + 下左；
- 制动：上下同侧；
- 滑行：全关。

PWM 调速：在切换时插入 PWM 占空比控制平均电压。

### 2.2 三相驱动器

3 个半桥（6 个开关）控制三相电机绕组。按转子位置切换通电顺序：
- 6 步方波（梯形波）；
- FOC（磁场定向控制）正弦波；
- 需要驱动芯片（如 DRV8301、FD6288）。

### 2.3 关键参数

- 死区时间：上下桥臂切换间隔（防直通）；
- PWM 频率：1 kHz~100 kHz；
- 电流采样：低边/高边采样；
- 反电动势/电流反馈：闭环控制用。

## 4. 关键参数与选型

| 参数 | 说明 |
|---|---|
| 工作电压 | V（max） |
| 输出电流 | A（持续/峰值） |
| Rds(on) | 导通电阻（mΩ），越低越高效 |
| PWM 频率 | kHz |
| 死区时间 | ns/µs |
| 逻辑电压 | 3.3V / 5V |
| 接口 | PWM + DIR / SPI / I2C / 模拟 |
| 保护 | 过流、过温、欠压、短路 |
| 散热 | 散热片/风扇需求 |
| 封装 | SOIC / QFN / 模块化 |

**选型建议**：
- 创客/小车：TB6612、L298N、DRV8871；
- 工业大电流：BTS7960、VNH2SP30、Cytron MD30C；
- BLDC：DRV8301 + MOSFET 模组；
- FOC：DRV8323 + MCU（STM32/ESP32）；
- 智能 BLDC：ODrive、VESC；
- 微型：STSPIN233、HR8833。


## 5. 硬件连接

- 电源：清洁、稳压、足够电流；
- 去耦电容：电机电源 100nF + 100µF；
- 自举电容：自举型半桥驱动（如 IR2104）需要 100nF 自举电容；
- 续流二极管：内置或外接（视型号）；
- 散热：大电流应用必须装散热片；
- **H 桥禁止直连**：必须先确认电源和地都接好再启动；
- 编码器接口：BLDC FOC 需 AB 编码器（部分支持无感）；
- 控制信号：逻辑电平兼容（3.3V/5V）；
- **死区时间必须足够**：否则上下桥臂直通短路，瞬间烧毁。



- RθJA 与功耗决定温升；
- PCB 覆铜散热（暴露铜面）；
- 散热片（铝）；
- 风扇（强制风冷）；
- 温升测试：实际负载下测量外壳温度（< 80°C）。


## 6. 通信方式

- **PWM + DIR**：简单（多数 H 桥）；
- **SPI**：智能驱动器配置寄存器；
- **I2C**：部分小型驱动器；
- **模拟电压**：少数 H 桥；
- **CAN/UART**：智能驱动器（VESC、ODrive）。


## 7. 初始化流程

### 7.1 H 桥

1. 配置 MCU GPIO（PWM + DIR）；
2. 设置初始为停止状态（PWM=0）；
3. 启动时缓慢升 PWM。

### 7.2 BLDC 驱动器

1. 配置预驱使能；
2. 配置栅极驱动电流；
3. 配置电流采样（分流电阻）；
4. 配置保护（过流阈值）；
5. 配置 PWM 频率和死区；
6. 使能输出。

### 7.3 FOC 驱动器

1. 测量电机参数（电阻、电感、极对数）；
2. 配置编码器；
3. 设置控制参数（PID）；
4. 校准电流和角度偏移；
5. 启动。


## 8. 驱动接口

```python
class HBridge:
    def __init__(self, pwm, dir_pin):
        self.pwm = pwm
        self.dir = dir_pin

    def begin(self):
        pass

    def forward(self, speed):
        self.dir.low()
        self.pwm.duty(speed)

    def backward(self, speed):
        self.dir.high()
        self.pwm.duty(speed)

    def brake(self):
        self.pwm.duty(100)  # 全占空比短路

    def coast(self):
        self.pwm.duty(0)

        pass

    def set_current(self, ia, ib):
        """设置相电流。"""
        pass

        self.en.high()
```

## 9. 数据格式与单位

- PWM：占空比 0~100% 或 0~65535；
- SPI：寄存器配置；
- 模拟：电压 V；
- FOC：dq 轴电流。


## 10. 错误处理

- 过流保护（OCP）：超过阈值关闭输出；
- 过温保护（OTP）：温度超限关闭；
- 欠压保护（UVP）：电源低关闭；
- 短路保护：桥臂直通立刻关闭；
- 反向 EMF 保护：使用续流回路；
- PWM 失效保护：信号线断开关闭。



- **驱动器报警**：读取报警寄存器/LED 闪烁；
- **电机不转**：PWM 未输出、H 桥未使能、电源未上电；
- **发热严重**：电流过大、散热不足；
- **H 桥直通短路**：死区时间不够、电源短路烧毁；
- **编码器错**：FOC 驱动器无法换相。


## 11. 低功耗

- 静态电流 < 100 µA（多数）；
- 关闭使能降低静态；
- 待机模式（部分智能驱动器）。


## 12. 示例代码

```python
hb = HBridge(pwm_ch, dir_pin)
hb.begin()
hb.forward(50)  # 50% 正转
time.sleep_ms(2000)
hb.brake()      # 制动
time.sleep_ms(500)
hb.backward(50)
time.sleep_ms(2000)
hb.coast()
```


## 13. 调试方法

1. 不接电机，测驱动器输出波形；
2. 低 PWM 启动电机（10%）观察；
3. 测量电流波形（示波器 + 电流探头）；
4. 测试正反转；
5. 长时间运行测温；
6. 测试过流保护。


## 14. 常见问题

- **电机不转**：使能未开、信号未输出、电源未上电；
- **电流过大**：机械卡死、加速过快；
- **驱动器过热**：散热不足、电流过大；
- **H 桥烧毁**：死区时间不足、桥臂直通；
- **FOC 启动失败**：电机参数不准、编码器接线错；
- **闭环振荡**：PID 参数不当、电流采样噪声大。
