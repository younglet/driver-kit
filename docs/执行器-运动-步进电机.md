# 步进电机


## 1. 模块概览

步进电机（Stepper Motor）将电脉冲转换为精确的角位移，开环控制即可获得高精度位置控制。常用于：
- 3D 打印机；
- CNC 雕刻机；
- 写字机、绘图仪；
- 相机云台；
- 自动化设备；
- 阀门控制；
- 贴片机；
- 实验室自动化。

类型：
- **永磁式（PM）**：永磁转子，步距大；
- **变磁阻式（VR）**：转子非永磁，扭矩小；
- **混合式（HB）**：综合两者，最常用；
- **单极性**：5/6 线，绕组中间抽头；
- **双极性**：4 线，全桥驱动。


## 2. 工作原理与适用场景

定子线圈按特定顺序通电，吸引转子齿对齐。改变通电顺序，转子按步距角转动。

- **整步（Full Step）**：每次通电 1 拍（单极）或 2 拍（双极）；
- **半步（Half Step）**：交替使用整步模式，步距减半；
- **微步（Microstep）**：细分电流，正弦波控制，步距更小（1/16、1/32、1/256）。

步距角常见：1.8°（200 步/转）、0.9°（400 步/转）。

## 5. 硬件连接

- 驱动：必须使用步进电机驱动器（详见电机驱动器文档）；
  - ULN2003（单极性 28BYJ-48）；
  - A4988 / DRV8825 / TMC2208 / TMC2209（双极性 NEMA17）；
- 接线：双极性 4 线（A+/A-/B+/B-）；
- 电源：恒流驱动（驱动器内置）；
- 散热：驱动器大电流运行需散热；
- 信号线：STEP（脉冲）+ DIR（方向）+ EN（使能）；
- 微步设置：MS1/MS2/MS3 跳线（TMC 通过 UART/SPI）；
- 防接错：A+/A- 接反会导致电机啸叫但不转。


## 6. 通信方式

- **STEP + DIR**：脉冲 + 方向信号（最常用）；
- **CW/CCW**：双脉冲正反转；
- **SPI**：TMC 系列配置；
- **UART**：TMC2209 单线配置。


## 7. 初始化流程

1. 接线（驱动器 + 电机）；
2. 设置微步（MS 跳线）；
3. 设置电流（Vref 调节）；
4. 启动电机（EN 引脚拉低）；
5. 发脉冲开始运动；
6. 配置加减速曲线。


## 8. 驱动接口

```python
class Stepper:
    def __init__(self, step_pin, dir_pin, en_pin=None):
        self.step = step_pin
        self.dir = dir_pin
        self.en = en_pin
        self.steps_per_rev = 200  # 1.8°

    def begin(self):
        if self.en:
            self.en.low()  # 使能
        self.dir.low()

    def move_steps(self, steps, delay_us=1000):
        """以脉冲方式转动指定步数。"""
        for _ in range(steps):
            self.step.high()
            time.sleep_us(delay_us)
            self.step.low()
            time.sleep_us(delay_us)

    def set_speed_rpm(self, rpm):
        """设置 RPM，根据 step delay 实现。"""
        step_delay_us = 60_000_000 // (self.steps_per_rev * rpm)
        self.step_delay = step_delay_us

    def rotate(self, degrees, direction=1):
        """旋转指定角度。"""
        steps = int(self.steps_per_rev * degrees / 360)
        if direction < 0:
            self.dir.high()
        else:
            self.dir.low()
        self.move_steps(steps)
```



启动/停止过快会失步，需要 S 曲线或梯形加减速：

```python
def trapezoidal_profile(distance, v_max, a):
    """返回 (t, position) 序列。"""
    t_accel = v_max / a
    d_accel = 0.5 * a * t_accel ** 2
    d_decel = d_accel
    d_const = distance - d_accel - d_decel
    if d_const < 0:
        # 三角形曲线
        t_accel = (distance / a) ** 0.5
        t_decel = t_accel
        t_const = 0
        # ...
    else:
        t_const = d_const / v_max
    return t_accel, t_const, t_decel
```



- 整步：每步全电流；
- 半步：相邻两相通 1/2 电流；
- 微步：1/4、1/8、1/16、1/32、1/256 电流比例；

## 9. 数据格式与单位

- 脉冲：数字信号（上升沿触发）；
- 方向：高低电平；
- 转速：RPM 或 PPS（Pulse Per Second）；
- 加速度：rad/s² 或 steps/s²；
- 电流：A。


## 10. 错误处理

- **失步**：加减速过快、扭矩不足、电流设置过低；
- **电机啸叫**：电流过大、A/B 相接错；
- **电机过热**：电流过大、散热不足；
- **振动**：低速共振、共振点避让；
- **驱动器报警**：过流、过温、短路。


## 11. 低功耗

- 停止时关闭电流（EN 引脚或待机模式）；
- TMC StealthChop 静态电流低；
- 减少保持扭矩（节能）。


## 12. 示例代码

```python
stepper = Stepper(step_pin, dir_pin, en_pin)
stepper.steps_per_rev = 200
stepper.begin()

stepper.set_speed_rpm(60)
stepper.rotate(360, direction=1)  # 正转 360°
time.sleep_ms(500)
stepper.rotate(-360, direction=-1)  # 反转 360°
```


## 13. 调试方法

1. 缓慢转动观察步距；
2. 测试微步平滑度；
3. 检查保持扭矩（断电能保持位置）；
4. 测试加减速（最大转速）；
5. 检查发热（长时间运行 < 70°C）；
6. 调节电流（Vref）。


## 14. 常见问题

- **电机不转**：接线错、信号未输出、驱动器未使能；
- **只振动不转**：A/B 相顺序错；
- **失步**：加减速过快、扭矩不足、负载过大；
- **发热严重**：电流设置过大；
- **高速失步**：提高供电电压（如 24V → 36V）、提高电流；
- **共振噪音**：避开低速共振点，或用微步。

## 16. 闭环控制

闭环步进（带编码器）：
- 不丢步（高负载加速时）；
- TMC 闭环；
- 编码器反馈位置；
- 价格高于开环。
