# 执行器总览

执行器（Actuator）是把电信号转化为物理动作的器件。DriverKit 收录 **11 个执行器模块**，按功能划分为 **运动电机、开关、流体、光源、声光** 五类。

## 按领域速查

### 运动电机（6 个）

| 模块 | 说明 | 文档 |
|---|---|---|
| 直流电机 | 有刷 DC，最基础、成本最低 | [打开](./执行器-运动-直流电机) |
| 无刷电机 | 电子换向、寿命长、效率高（无人机主流） | [打开](./执行器-运动-无刷电机) |

## 共性知识

- **必须搭配驱动器**：电机不能直接接 MCU
- **续流二极管 / 死区时间**：H 桥防直通与感性负载保护
- **散热决定寿命**：功率器件温升 < 80°C
- **闭环 vs 开环**：精密应用选编码电机 + 闭环控制
- **PWM 频率**：电机 1~25 kHz，LED > 200 Hz 避免闪烁
