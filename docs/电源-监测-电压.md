# 电压监测


## 1. 模块概览

电压监测（Voltage Monitoring）是嵌入式系统最常用的电源感知功能，用于：
- 电池电压监测；
- 电源故障检测；
- 低压保护；
- 能量收集；
- USB Type-C 电压检测；
- 太阳能充电监测；
- 多路电源切换；
- ADC 测量。

类型：
- **电阻分压 + MCU ADC**：低成本；
- **专用 ADC 芯片**：高精度；
- **专用电源监测芯片**：带复位 + 中断；
- **电流/功率监测芯片**（如 INA219）：多参数；
- **电压参考芯片**：精密源；
- **数字电压表模块**：成品模块。


## 2. 工作原理与适用场景

### 2.1 电阻分压

- 高电压 → 电阻分压 → MCU ADC 量程内；
- Vadc = Vin × R2 / (R1 + R2)；
- 选择 R1、R2 比例，使最大电压对应 ADC 满量程；
- 加 RC 滤波（消除噪声）；
- 加齐纳二极管（限压保护）。

### 2.2 ADC 分辨率

- 8-bit（0.4% / 256 级）；
- 10-bit（0.1% / 1024 级）；
- 12-bit（0.024% / 4096 级）；
- 16-bit（0.0015% / 65536 级）；
- 24-bit（高精度测量）；

### 2.3 电压参考

- **MCU 内部参考**：1.1V/2.5V/3.0V（精度低，±5%）；
- **外部参考**：TL431、REF5025、ADR01/02/05；
- **高精度 ADC 内置参考**：如 ADS1220（2.048V 内部参考）；
- 选择影响测量精度。

## 5. 硬件连接

### 5.1 电阻分压

- R1（高压端）、R2（低压端）分压；
- 分压比 = R2 / (R1 + R2)；
- R1 + R2 通常 10 kΩ~1 MΩ（越大电流越小，但噪声越大）；
- 0.1% 精度电阻获得高精度（但成本高）；
- ADC 输入阻抗 >> R2（避免分压误差）。

### 5.2 滤波

- 100nF + 10kΩ RC 低通；
- 截止频率 ~1.6 kHz；
- 滤除高频噪声。

### 5.3 保护

- TVS 二极管（防瞬态电压）；
- 串联电阻（限流）；
- 齐纳二极管（限压）；
- 输入电容（滤波尖峰）。

### 5.4 高电压隔离

- 隔离运放（如 ACPL-C870）；
- 电压互感器（ZMPT101B）；
- 差分放大器。


## 6. 通信方式

- 模拟：电阻分压直接 ADC；
- I2C：ADS1115、INA219、INA226；
- SPI：MCP3008、ADS1220、ADS1256；
- 中断：阈值中断（如低电压）。

```python
class VoltageSensor:
    """电阻分压 + MCU ADC。"""
    def __init__(self, adc, r1, r2):
        self.adc = adc

    def read_voltage(self):
        """返回实测电压 V。"""
        v_adc = self.adc.read_voltage()

class BatteryMonitor:
    """电池监测（分压 + ADC + 库仑计）。"""
    def __init__(self, adc):
        self.adc = adc

    def voltage(self):
        """返回电池电压 V。"""
        return self.adc.read_voltage() * self.scale

    def percentage(self):
        """返回电量百分比（基于电压）。"""
        v = self.voltage()
        if v > 4.2:
            return 100
        elif v < 3.0:
            return 0
        else:
            return (v - 3.0) / (4.2 - 3.0) * 100

    def low_battery(self):
        """低电量报警。"""
        return self.voltage() < 3.3
```



锂电池放电曲线：
- 4.2V：100%（充满）；
- 3.85V：~50%；
- 3.6V：~20%；
- 3.3V：~5%；
- 3.0V：0%（建议停止放电）。

具体关系取决于负载和电池特性，建议查电池厂家曲线。


## 9. 数据格式与单位

- 电压：V 或 mV；
- 原始值：ADC 计数；
- 电池百分比：0~100%；
- 阈值：触发动作。


## 10. 校准与滤波

### 10.1 增益校准

- 用已知电压源校准（如 3.000V 参考）；
- 计算实际分压比；
- 校正系数。

### 10.2 偏移校准

- 输入接地读 0；
- 计算偏移值。

### 10.3 校准流程

```python
def calibrate(adc, ref_voltage):
    raw = adc.read_raw()
    v = adc.read_voltage()
    return ref_voltage / v  # 增益校正系数
```



- 中值滤波去除尖峰；
- 滑动平均（10~100 样本）；
- 加权滑动平均（高权重新值）；
- 低通滤波（RC 软件滤波）。


## 11. 中断与 DMA

- 低电压中断（电池低压保护）；
- 高电压中断（过压保护）；
- 阈值中断（电阻分压 + 比较器）；
- ADC 转换完成中断。


## 12. 错误处理

- **ADC 错误**：I2C/SPI 通信失败、读 0xFFFF；
- **校准失败**：参考电压不准；
- **分压电阻失效**：焊点脱开、电阻损坏；
- **输入超量程**：限压二极管；
- **EMI 干扰**：加滤波、PCB 走线优化。


## 13. 低功耗

- ADC 低功耗模式（µA）；
- 间歇测量（每分钟 1 次足够电池监测）；
- 关闭 ADC；
- 阈值中断唤醒 MCU。

## 15. 调试方法

1. 测分压电阻两端电压；
2. 测 ADC 输入电压；
3. 用万用表对照；
4. 测电源纹波；
5. 校准验证（参考电压）；
6. 测试低电压保护。


## 16. 常见问题

- **测量不准**：分压电阻误差、ADC 精度、参考电压；
- **低电压不报警**：阈值设置错、中断未启用；
- **纹波大**：电源不稳，加滤波；
- **读数跳变**：EMI、滤波不足；
- **ADC 烧毁**：输入超量程；
- **电池百分比不准**：负载电流影响电压，需考虑库仑计。


## 17. 参考资料

- ADC 输入阻抗必须远大于 R2；
- 高电压必须分压（不能直接进 MCU）；
- 加保护电路（TVS、齐纳）；
- 校准使用精密参考；
- 电池电量百分比只是估算（与电流、温度相关）；
- 工业应用需要隔离运放。
