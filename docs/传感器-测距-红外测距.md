# 红外测距传感器


## 1. 模块概览

红外测距传感器通过红外 LED 发射调制光，接收反射光强度或角度，估算距离。常见于：
- 机器人避障；
- 物体接近检测；
- 自动门感应；
- 液位检测；
- 黑线循迹；
- 工业产品计数。

类型：
- **三角测量式（Sharp GP2Y 系列）**：PSD 位置敏感元件，输出模拟电压；
- **反射强度式**：红外发射+接收对管；
- **TOF 式**：见 `激光测距.md`；
- **反射式光电开关**：对管，单点检测。


## 2. 工作原理与适用场景

### 2.1 三角测距（Sharp）

红外 LED 发射光束打到目标，反射光通过透镜打到 PSD（位置敏感元件）。不同距离反射点位置不同，按三角函数计算距离。输出与距离呈非线性曲线（近距陡增、远距趋平）。

### 2.2 反射强度式

红外发光 → 反射回光电二极管 → 信号强度与距离/反射率相关。
- 黑色物体反射弱、距离读数偏远；
- 白色物体反射强、距离读数偏近；
- 输出非线性，难以精确测距。

**选型建议**：
- 10~80 cm 通用避障：GP2Y0A21YK0F；
- 远距：GP2Y0A02YK0F（20~150 cm）；
- 近距：GP2Y0A41SK0F（4~30 cm）；
- I2C：GP2Y0E02/03；
- 简易避障：FC-03 + 数字输出；
- 精确测距：考虑 TOF（激光文档）。


## 5. 硬件连接

- 模拟输出：ADC 采样，注意电压范围（Sharp 输出 0.4V~3.1V）；
- 数字输出：直接 GPIO 检测高低电平；
- I2C：GP2Y0E 系列默认 0x40；
- 滤除环境光干扰：Sharp 内置调制（调制频率 25 kHz）；
- 注意：对阳光和红外光敏感（直对阳光会饱和）；
- 反射物体：白色最好，黑色误差大；
- 安装：与物体表面正对，倾角 < 5°。


## 6. 通信方式

- 模拟：电压输出；
- 数字：高低电平（阈值由硬件比较器或 MCU 程序判定）；
- I2C：GP2Y0E 系列支持（GP2Y0E02/03）；
- GP2Y0D 系列：数字 TTL 输出，阈值固定。


## 7. 初始化流程

1. 配置 ADC（模拟）或 GPIO/I2C（数字）；
2. 上电等待稳定（约 30ms）；
3. 周期性采样；
4. 数字输出直接读取；
5. 模拟输出查表或拟合曲线换算距离。

```python
    """v 0.0~3.1 V → cm"""
```

## 9. 数据格式与单位

- 模拟输出：电压（V）；
- 距离：cm；
- 数字输出：高低电平；
- 单位换算：通过查表或拟合公式。


## 10. 校准与滤波

Sharp 系列为非线性，典型曲线：
- 10 cm → 2.3 V
- 20 cm → 1.1 V
- 50 cm → 0.5 V
- 80 cm → 0.4 V

工程中常用拟合公式：

```python
# GP2Y0A21YK0F
def gp2y0a21_21(v):
    if v <= 0.0:
        return 80.0
    return 29.98 / (v + 0.1) - 5.0
```



- 中值滤波去除尖峰；
- 滑动平均降低抖动；
- 跳过 0.4V 以下电压（远距饱和区）；
- 跳过 3.1V 以上（盲区）。


## 11. 中断与 DMA

- 数字输出可用于中断唤醒；
- 模拟采样周期 30~50 ms 足够。


## 12. 错误处理

- 距离饱和：远距或强反射（户外阳光）；
- 数字输出不稳：阈值模糊，使用中值滤波；
- 距离偏小：黑色物体反射弱；
- 模拟读数 0：电源未上电。


## 13. 低功耗

- Sharp 模块约 30 mA；
- 间歇测量：1 Hz 足够大部分应用；
- 数字输出模块可在无目标时降频。


## 14. 示例代码

```python

    def __init__(self, adc):
        self.adc = adc

    def read_voltage(self):
        return self.adc.read_voltage()

    def read_distance_cm(self):
        v = self.read_voltage()
        if v < 0.4:
            return 80.0
        return 29.98 / (v + 0.1) - 5.0
```

## 15. 调试方法

1. 测试白色墙面 10~80 cm 不同距离读数；
2. 对比黑色/白色物体差异；
3. 室外阳光下测试（注意饱和）；
4. 检查波形（调制光 25 kHz）。


## 16. 常见问题

- **Sharp 输出接近 0.4V**：远距或无目标；
- **数字输出频繁跳变**：阈值不稳，使用硬件比较器或软件滤波；
- **距离偏小/偏大**：反射率差异，需要相对位置而非绝对距离；
- **阳光下饱和**：加遮光罩；
- **TOF 选哪个**：若精度要求高，建议看 ToF（VL53L 系列）。
