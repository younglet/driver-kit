# 传感器总览

传感器是嵌入式系统感知世界的窗口。DriverKit 收录 **17 个传感器模块**，按测量对象划分为 **位姿、环境、测距、接近、视觉** 五个领域。

## 按领域速查

### 位姿（6 个）

测量加速度、角速度、角度、磁场等运动与方向参数。

| 模块 | 说明 | 文档 |
|---|---|---|
| 三轴陀螺仪 | 测量绕 X/Y/Z 轴的角速度 | [打开](./传感器-位姿-三轴陀螺仪) |
| 三轴加速度计 | 测量 X/Y/Z 轴加速度（含重力分量） | [打开](./传感器-位姿-三轴加速度计) |
| 六轴 IMU | 加速度计 + 陀螺仪，姿态融合 | [打开](./传感器-位姿-六轴IMU) |
| 九轴 IMU | 六轴 + 磁力计，提供绝对航向 | [打开](./传感器-位姿-九轴IMU) |
| 磁力计 | 测量三维地磁场，电子罗盘 | [打开](./传感器-位姿-磁力计) |
| 编码器 | 增量式/绝对值旋转与角度测量 | [打开](./传感器-位姿-编码器) |

## 共性知识

- **I2C 是绝对主流**（多数现代传感器接口）
- **校准不可省**：加速度计陀螺仪需零偏校准，磁力计需硬铁/软铁校准
- **坐标轴定义以数据手册为准**，不同厂商不同方向
- **温度漂移影响精度**，高精度场景需温度补偿
