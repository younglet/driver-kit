# 电流监测


## 1. 模块概览

电流监测（Current Monitoring）是测量电路中的电流消耗或电流流向，常用于：
- 电池容量估算（库仑计）；
- 电机电流监测；
- 功率测量；
- 过流保护；
- 太阳能电池监测；
- 电源调试；
- 故障诊断；
- 工厂能源管理。

类型：
- **采样电阻 + 运放**：经典方案；
- **霍尔传感器**：隔离测量；
- **专用电流监测芯片**（INA219/INA226）：集成度高；
- **电流互感器（CT）**：AC 大电流测量；
- **库仑计芯片**：电池专用；
- **分流器（Shunt）**：大电流场景。


## 2. 工作原理与适用场景

### 2.1 采样电阻（低边/高边）

- 串联一个已知阻值的精密电阻（毫欧级）；
- 测量电阻两端电压（mV 级）；
- I = V / R；
- **低边测量**：电阻在负载和 GND 之间（简单，但 GND 不共地）；
- **高边测量**：电阻在电源和负载之间（不破坏 GND，共地）；
- **运放放大**：放大 mV 信号到 ADC 量程。

### 2.2 霍尔传感器

- 利用霍尔效应，测量通过电流产生的磁场；
- 隔离测量（无电气接触）；
- ACS712 输出与电流成比例的电压；
- INA180 系列内置分流电阻+运放。

### 2.3 专用芯片

- INA219 / INA226 / INA228：内置 ADC + 运放，I2C 输出；
- 同时测电压、电流、功率；
- 精度可达 ±0.1%。

## 5. 硬件连接

### 5.1 采样电阻

- 低边（简单但 GND 不共）；
- 高边（保留 GND 共地）；
- 电阻精度 ±1% 或更高；
- 低温度系数（TCR < 50 ppm/°C）；
- 4 线连接（开尔文）减少引线误差；
- 功率：P = I² × R（如 10 A × 10 mΩ = 1 W）；
- 散热设计。

### 5.2 运放放大

- 增益：100~200 合适；
- 失调电压 < 50 µV；
- 偏置电流小；
- 共模抑制比 > 100 dB。

### 5.3 霍尔传感器

- 模块供电 3.3V/5V；
- 输出电压与电流成比例；
- 中心电压 = VCC / 2（双向）；
- 注意共模电压范围；
- 模块工作电压不超过额定值。

### 5.4 滤波

- 低通 RC 滤波（带宽限制）；
- 软件中值/平均滤波；
- 去除开关电源噪声。


## 6. 通信方式

- 模拟：直接 ADC；
- I2C：INA219 / INA226 / MAX17048；
- SPI：INA238 / ADS131；
- 数字：部分芯片内置 ADC 输出数字值。

## 8. 驱动接口

```python
class CurrentSensor:
    """采样电阻 + 仪表放大器 + ADC。"""
    def __init__(self, adc, shunt_ohm, gain=100, vref=3.3):
        self.adc = adc
        self.shunt = shunt_ohm
        self.gain = gain
        self.vref = vref

    def read_current(self):
        """返回电流 A。"""
        v = self.adc.read_voltage()
        # v = i × shunt × gain
        i = v / (self.shunt * self.gain)
        return i


class ACS712:
    """ACS712 霍尔电流传感器。"""
    def __init__(self, adc, sensitivity_mv_a=185, vcc=5.0):
        self.adc = adc
        self.sens = sensitivity_mv_a / 1000  # V/A
        self.zero = vcc / 2  # 中心电压

    def read_current(self):
        v = self.adc.read_voltage()
        return (v - self.zero) / self.sens
```

库仑计（Charge Counting）通过积分电流估算剩余电量：

```python
class CoulombCounter:
    def __init__(self, current_sensor, dt_seconds=1):
        self.cs = current_sensor
        self.dt = dt_seconds
        self.accumulated = 0  # mAh

    def update(self):
        i_mA = self.cs.read_current() * 1000  # mA
        # mAh = mA × 小时
        self.accumulated += i_mA * self.dt / 3600

    def remaining_mah(self, initial_mah):
        return initial_mah - self.accumulated

    def percentage(self, initial_mah):
        return max(0, self.remaining_mah(initial_mah) / initial_mah * 100)
```


## 9. 数据格式与单位

- 电流：A 或 mA；
- 功率：W 或 mW；
- 电压：V；
- 电量：% 或 mAh；
- 分流电压：mV。


## 10. 校准与滤波

### 10.1 采样电阻校准

- 用万用表测实际电阻值；
- 记录精确值用于软件计算。

### 10.2 零点校准

- 无电流时读 0；
- 计算零点偏移。

### 10.3 增益校准

- 用已知电流源校准；
- 调整缩放系数。

### 10.4 温度补偿

- 电阻 TCR 影响；
- 实时温度补偿（部分芯片内置）。



- 中值滤波：去除尖峰；
- 滑动平均：低通特性；
- α-β 滤波器：状态估计；
- 卡尔曼滤波：最优估计。


## 11. 中断与 DMA

- 过流中断（INA226 等内置）；
- 阈值中断（ALERT 引脚）；
- 软件比较。


## 12. 错误处理

- **读数跳动**：滤波不足、采样电阻接触不良；
- **零点偏移**：温度漂移；
- **量程不足**：电流超出量程；
- **过载烧毁**：电流超出电阻功率；
- **EMI 干扰**：开关电源噪声。


## 13. 低功耗

- ADC 低功耗模式；
- 间歇测量；
- 唤醒时测量（睡眠时关闭）；
- INA226 待机 < 1 µA。

## 15. 调试方法

1. 用万用表测分流电阻电压（验证）；
2. 用示波器看波形（开关电源纹波）；
3. 用电流表串联验证；
4. 测共模电压；
5. 检查零点偏移；
6. 测试过流保护。


## 16. 常见问题

- **电流读数不准**：分流电阻误差、运放失调、温漂；
- **读数跳动**：开关电源噪声、滤波不足；
- **零点漂移**：温度变化，重新校准；
- **量程不足**：电阻太小或太大；
- **测量失真**：共模电压超限；
- **烧毁**：过载或反接。


## 17. 参考资料

- 高边测量更安全（保留共地）；
- 大电流需考虑散热；
- 共模电压不要超出 ADC 量程；
- 加 TVS 防浪涌；
- 4 线（开尔文）连接减小误差；
- 工业级隔离（AC、高压）。
