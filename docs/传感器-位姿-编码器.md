# 编码器


## 1. 模块概览

编码器（Encoder）是一种位置/角度传感器，将机械位移（旋转或线性）转换为电信号。常见于电机伺服、机器人关节、流水线、数控机床、3D 打印机。

主要类型：
- **增量式编码器**：输出 A/B 相脉冲，旋转时产生计数；
- **绝对值编码器**：每个位置对应唯一编码，掉电不丢失；
- **线性编码器**：测量直线位移；
- **光学编码器**：光栅 + 光电元件；
- **磁性编码器**：霍尔/AMR/TMR 检测磁环；
- **电容编码器**：电容变化检测。


## 2. 工作原理与适用场景

### 2.1 增量式

内部有均匀刻度的光栅/磁环，旋转时输出两路正交方波 A/B 相，相位差 90°。旋转方向通过 A 相对 B 相的先后判定。每转产生 N 个脉冲（PPR / CPR）。

常见应用：
- 直流电机测速；
- 步进电机闭环；
- 手动旋钮（音量）；
- 数控机床主轴；
- 输送带测距。

### 2.2 绝对值

每个机械位置对应唯一码值（灰码或二进制）。接口有 SPI、SSI、BISS、RS485、CAN 等。

应用：
- 机器人关节（精度高、不掉电丢失）；
- CNC 机床；
- 工业自动化。

### 2.3 磁性编码器

磁铁安装在旋转轴上，磁敏感元件（霍尔/AMR/TMR）检测磁场方向。结构紧凑、抗污染。

应用：电机内嵌（BLDC、PMSM）、机器人关节、无人机云台。

## 4. 关键参数与选型

| 参数 | 说明 |
|---|---|
| 分辨率/CPR | 每转脉冲数（增量）或位数（绝对） |
| 精度 | 弧分、角秒 |
| 接口 | 增量 AB 相 / SSI / BISS / SPI / I2C / RS485 / CAN |
| 工作电压 | 3.3V / 5V / 24V |
| 最大转速 | rpm |
| 防护等级 | IP 等级 |
| 工作温度 | -40°C ~ +85°C |
| 轴类型 | 实心轴 / 空心轴 / 轴套 |
| 抗冲击振动 | g / Hz |

**选型建议**：
- 旋钮/小项目：EC11 / EC12（机械式）/ AS5600（磁性）；
- 电机伺服：增量式 AB 相 + 索引 Z；
- 机器人关节：高精度绝对值（AS5048A、AEAT、RLS）；
- 工业 CNC：光电编码器 + 直线尺。


## 5. 硬件连接

- AB 相差分输出：使用 RS422 接收器（如 AM26LS32、MAX3094）；
- 电源去耦：100 nF + 10 µF；
- 屏蔽双绞线传输，降低 EMI；
- 磁性编码器：磁铁与芯片对齐（径向充磁 vs 轴向充磁）；
- 屏蔽电缆：长距离使用屏蔽线；
- 接地：单点接地避免环路。


## 6. 通信方式

- 增量 AB 相：方波信号，硬件定时器捕获；
- 增量 + 索引 Z：每转一个零位信号；
- SSI（同步串行）：CLK + DO，单向绝对值；
- BISS：SSI 升级版，支持双向通信和参数读取；
- SPI：部分绝对值芯片支持；
- I2C：AS5600、AS5601 等；
- 模拟/PWM：低成本方案。


## 7. 初始化流程

1. 配置 I2C（默认地址 0x36）；
2. 验证 WHO_AM_I（如有）；
3. 设置零点（写 ZPOS 寄存器）；
4. 设置最大角度（MPOS）；
5. 设置输出模式（ANG/RAW）；
6. 配置 PWM 频率（如使用 PWM 输出）；
7. 开始读取角度。

### 10.1 M 法（测频）

固定时间窗内数脉冲数。低速时精度差。

```python
pulses = encoder.read_pulses()
rpm = pulses * 60 / (cpr * dt)
```

### 10.2 T 法（测周）

测量相邻脉冲时间间隔。高速时精度差。

### 10.3 M/T 法

结合两者，宽速度范围精度好。


## 9. 数据格式与单位

- 增量：脉冲数 / CPR；
- 绝对：位数（10/12/14/16/18/20/24 bit）；
- 角度单位：度（°）、弧度（rad）、计数（count）；
- 角速度单位：rpm（转每分钟）、dps（度每秒）；
- 数据范围：0~360° / -180°~+180° / 0~2π。


## 10. 校准与滤波

- 中值滤波：去除脉冲干扰；
- 滑动平均：降低速度抖动；
- 卡尔曼滤波：状态估计（角度 + 速度）。


## 11. 中断与 DMA

- 增量式：硬件定时器捕获中断；
- 高速应用：DMA + 定时器；
- 绝对值：按需读取。


## 12. 错误处理

- 编码器断线：检查 AB 相电平；
- 零位丢失：电池掉电导致绝对值编码器位置丢失；
- 计数溢出：32-bit 计数；
- 信号干扰：屏蔽线 + 滤波电容。


## 13. 低功耗

- 编码器本身一般无低功耗；
- 主机可关闭定时器中断减少唤醒。


## 14. 示例代码

```python
class Encoder:
    def __init__(self, cpr=360):
        self.cpr = cpr
        self.count = 0

    def begin(self):
        pass

    def update(self, delta):
        """由中断调用：累加计数。"""
        self.count += delta

    def read_angle(self):
        """返回 0~360°。"""
        return (self.count % self.cpr) * 360.0 / self.cpr

    def read_velocity(self, dt):
        """返回 rpm。"""
        return self.count / self.cpr * 60 / dt

    def reset(self):
        self.count = 0


enc = Encoder(cpr=360)
enc.begin()
while True:
    deg = enc.read_angle()
    print("angle=%.1f°" % deg)
```


## 15. 调试方法

1. 手动旋转观察计数变化；
2. 慢速匀速旋转观察速度平稳性；
3. 快速正反转观察方向；
4. 检查 AB 相信号波形（示波器）；
5. 验证零位 Z 索引信号。


## 16. 常见问题

- **方向反**：交换 A/B 接线或软件 reverse；
- **计数跳变**：机械振动抖动，加硬件滤波或软件去抖；
- **零位丢失**：绝对值编码器电池电压低；
- **精度不足**：线数低、采样抖动、磁铁位置偏差。
