# 温湿度传感器


## 1. 模块概览

温湿度传感器测量环境温度和相对湿度（RH%），常见于气象站、空调、农业大棚、智能家居、仓库监测、医疗等。

类型：电阻式、电容式、半导体式。电容式最常用（DHT 系列、SHT 系列、BME 系列）。


## 2. 工作原理与适用场景

- **湿敏电容**：感湿膜吸附水分子导致介电常数变化，电容值随之改变；
- **温度传感器**：硅带隙温度传感器或热敏电阻；
- **露点计算**：通过温度和湿度算露点温度，避免冷凝。

应用：气象监测、智能家居 HVAC、温室大棚、博物馆/档案馆温控、机房、药品仓储、个人气象站。

## 4. 关键参数与选型

| 参数 | 说明 |
|---|---|

| 温度精度 | ±0.1°C（顶级）~ ±2°C（入门） |
| 湿度范围 | 0~100% RH |
| 湿度精度 | ±1.5%（顶级）~ ±5%（入门） |
| 分辨率 | 温度 0.01°C，湿度 0.01% RH |
| 响应时间 | 慢（5~30s）到快（2~8s） |
| 长期漂移 | <0.5%/年（好）~ 1%/年（一般） |
| 接口 | I2C、单总线（OneWire）、SPI |
| 电源 | 1.8V~3.6V / 3V~5V |
| 功耗 | 测量 mA 级，待机 µA 级 |

**选型建议**：
- 入门：DHT11、DHT22；
- 高精度：SHT31、SHT35、SHT45、BME280；
- 工业级：SHT35、SHT45、HDC2010；
- 国产替代：AHT20、AM2320，性价比高；
- 单总线方案：DHT22、DS18B20；
- 三合一：BME280（温湿压）。


## 5. 硬件连接

- DHT11/22 单总线：DATA 引脚需 4.7kΩ 上拉到 VCC；
- I2C 上拉：4.7kΩ；
- 远离发热体：传感器自身不发热，但会被 PCB 加热；
- 防冷凝：湿度传感器冷凝后可能损坏；
- 防水型号：带 PTFE 滤膜（SHT3x 防水版）；
- I2C 地址：BME280 默认 0x76（SDO=高 0x77），SHT3x 默认 0x44（ADDR 高 0x45）。


## 6. 通信方式

### 6.1 单总线（DHT11/22）

主机发送开始信号（拉低 18ms）→ 传感器响应（拉低 80us，拉高 80us）→ 40 bit 数据（5 字节：湿度整数、小数、温度整数、小数、校验和）。

### 6.2 I2C

标准 100/400 kHz，时序简单。

### 6.3 SPI

少数芯片支持（如 BME280）。


## 7. 初始化流程

1. 配置 I2C 总线；
2. 读取 WHO_AM_I（如有）；
3. 软复位；
4. 设置测量模式（单次 / 连续）；
5. 设置精度档位（高/中/低）；
6. 设置测量参数（温度、湿度、压力）；
7. 触发测量；
8. 等待数据就绪；
9. 读取补偿值（如 SHT4x）。

## 9. 数据格式与单位

- 温度：°C；
- 湿度：%RH（0~100%）；
- 露点：°C，由温度和湿度计算：`td = (b * γ(T, RH) + c) / (a - γ(T, RH))`；
- 字节序：MSB 在前；
- 数据更新周期：单次测量 10~15ms，连续测量按 ODR。


## 10. 校准与滤波

- 大部分芯片出厂已校准；
- 高精度需求：用标准温湿度源（如恒温水槽、盐溶液）校准；
- 校准值保存版本号 + CRC32；
- 长期使用每年漂移 0.5~1% RH，建议定期校准。


## 11. 错误处理

- 校验和错误（DHT 系列常见）：时序未严格遵守，重试；
- I2C 超时：检查地址、上拉；
- 数据饱和：温度/湿度超量程；
- 露水凝结：传感器泡水需干燥或更换；
- 长期漂移：定期校准。


## 12. 低功耗

- 单次测量后自动休眠；
- 测量电流 ~0.5mA，休眠 < 1µA；
- 可使用间歇唤醒（1 秒 1 次）做电池供电。

## 14. 调试方法

1. 检查芯片引脚电压；
2. 读取 WHO_AM_I 验证通信；
3. 室内常温比对：手持传感器观察温度上升；
4. 哈气观察湿度上升；
5. 冰水比对 0°C，加热水比对 100°C；
6. 露点温度与天气 App 比对。


## 15. 常见问题

- **DHT22 总是失败**：单总线时序精度要求高（µs 级），用定时器或 DWT 延时；
- **DHT11 数据全 0**：上拉电阻未接；
- **湿度不准**：高湿环境存放时间过长，恢复时间可能需要数小时；
- **温度偏高**：芯片被 PCB 加热，移至通风处；
- **I2C 错误**：地址错（DHT 系列没有 I2C）。

## 17. 露点和绝对湿度

```python
import math

def dew_point(t, rh):
    """Magnus 公式近似"""
    a, b, c = 17.27, 237.7, 17.5
    gamma = (a * t / (b + t)) + math.log(max(rh / 100.0, 1e-6))
    return b * gamma / (a - gamma)
```

绝对湿度 (g/m³) = 216.7 × (RH/100) × 6.112 × exp(17.62 × T / (243.12 + T)) / (273.15 + T)
