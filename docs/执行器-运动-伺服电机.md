# 伺服电机


## 1. 模块概览

伺服电机（Servo Motor）是一种高精度闭环电机系统，集成了电机 + 编码器 + 驱动器，常用于：
- 工业机器人关节；
- CNC 机床主轴/进给；
- 印刷包装机械；
- 半导体设备；
- 自动化生产线；
- AGV/AMR 驱动；
- 医疗设备；
- 高端 CNC/激光设备。

类型：
- **交流伺服**：主流，工业级；
- **直流伺服**：早期，现较少；
- **永磁同步伺服（PMSM）**：主流；
- **异步伺服**：低成本场合；
- **步进伺服**：步进电机 + 编码器（低成本高精度）；
- **直线伺服**：直线运动。


## 2. 工作原理与适用场景

伺服系统由伺服驱动器 + 伺服电机组成：
- 电机（永磁同步 PMSM 或异步）；
- 高精度编码器（17~23 bit，每转 131072~8388608 脉冲）；
- 驱动器内置 PID/FOC 控制；
- 接收位置/速度/扭矩指令，闭环输出。

控制模式：
- **位置模式**：精确角度/距离控制；
- **速度模式**：恒定转速；
- **扭矩模式**：恒定扭矩输出；
- **混合模式**：位置+扭矩、速度+扭矩等。

| 参数 | 说明 |

| 通信 | 脉冲+方向 / RS485 / CANopen / EtherCAT / MECHATROLINK |
| 反馈精度 | ±1 脉冲（高编码器） |
| 速度响应频率 | 1kHz+ |
| 工作温度 | 0~40°C（驱动器）/ 0~55°C（电机） |

**选型建议**：
- 入门/小型机械：台达 ASDA-B2、汇川 IS620N、步进伺服；
- 中型机械：汇川 SV660、台达 ASDA-A2、安川 Σ-7；
- 高端/CNC：三菱 J4、安川 Σ-X、西门子 S120；
- 总线控制：EtherCAT、CANopen；
- 脉冲控制：低成本方案。


## 4. 硬件连接

- 三相电源：220V AC 单相 / 380V AC 三相（按驱动器规格）；
- 控制电源：DC 24V（部分驱动器需要）；
- 编码器线：双绞屏蔽，阻抗匹配；
- 电机动力线：粗线（如 1.5~6 mm²），短距离；
- 接地：PE 接地必要；
- 制动电阻：制动能量大时外接；
- 抗干扰：编码器线屏蔽、与动力线分开走线。


## 5. 通信方式

### 6.1 脉冲+方向（传统）

- PUL（脉冲）+ DIR（方向）；
- 差分（RS422）：抗干扰；
- 单端：低成本；
- 脉冲频率：500 kHz~4 MHz。

### 6.2 总线通信

- **Modbus RTU**：通用协议；
- **CANopen**：欧洲主流；
- **EtherCAT**：高速实时（μs 级）；
- **MECHATROLINK**：日本主流；
- **PROFINET/PROFIBUS**：西门子系统；
- **RS485/RS422**：低成本总线。

### 6.3 模拟量

±10V 速度/扭矩给定。


## 6. 初始化流程

1. 接线（电源、电机、编码器、控制信号）；
2. 上电；
3. 设置电机参数（型号、编码器极对数）；
4. 设置控制模式（位置/速度/扭矩）；
5. 设置 PID 参数（自动调整或手动）；
6. 设置输入信号（脉冲 + 方向 / 总线）；
7. 设置电子齿轮比；
8. 保存参数；
9. 试运行。

设置电机转 1 圈对应的指令脉冲数：

```text
电机 1 圈 = 编码器 P/R × 4 倍频 = 131072 × 4 = 524288（17bit）
电子齿轮比 = 524288 / 用户期望每转脉冲数
```

如 1 万脉冲每转：524288 / 10000 ≈ 52.4。



- 自动调整：很多驱动器有"自动"按钮（一键调谐）；
- 手动调整：先调速度环 PI，再调位置环 P；
- 刚性等级：低刚性（柔性）~ 高刚性（刚性）；
- 惯量比：负载/电机惯量比，< 30 较易调整。


## 8. 数据格式与单位

- 脉冲：方波信号（脉冲+方向）；
- Modbus RTU：寄存器读写（功能码 0x06/0x03/0x10）；
- 位置单位：脉冲数（需按编码器位数换算）；
- 速度：rpm；
- 扭矩：% 或 N·m。


## 9. 错误处理

- **过流**：机械卡死或电机短路；
- **过载**：长时间超额定扭矩；
- **编码器错误**：编码器断线、噪声；
- **欠压/过压**：电源异常；
- **超速**：超出最高转速；
- **失步**（步进伺服）：负载过大；
- 报警代码：查看驱动器手册；
- 复位方法：清除报警 + 重新使能。



- 伺服系统能量大，必须有急停（E-stop）；
- 编码器断线可能引起不可控运动，需软件监控；
- 试运行先空载，再轻载，最后重载；
- 防止冲程（机械限位）。


## 10. 低功耗

- 待机时关闭使能；
- 制动时能量回收（部分驱动器）；
- 驱动器空载损耗。


## 11. 示例代码

```python
servo = Servo(uart, address=1)
servo.begin()
servo.set_mode(0)  # 位置模式

servo.enable()
servo.write_position_pulse(0)
time.sleep_ms(100)
servo.write_position_pulse(10000)
time.sleep_ms(500)
servo.write_position_pulse(0)
time.sleep_ms(500)

servo.disable()
```


## 12. 调试方法

1. 上电确认无报警；
2. JOG 模式点动测试；
3. 自动调谐（PID）；
4. 负载测试（轻载~重载）；
5. 跟随误差（位置偏差）；
6. 长时间运行温度检查；
7. 编码器波形测试。


## 13. 常见问题

- **电机不转**：使能未开、报警未清、控制信号错；
- **跟随误差大**：PID 不当、电子齿轮比错、刚性不足；
- **振动噪音**：PID 参数不当、惯量比过大；
- **过载报警**：机械卡死、负载过大；
- **编码器报警**：编码器线缆故障、干扰；
- **总线通信失败**：地址错、波特率不匹配、终端电阻缺失；
- **失控/飞车**：控制信号接错、伺服参数错。
