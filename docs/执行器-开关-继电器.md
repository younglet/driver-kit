# 继电器


## 1. 模块概览

继电器（Relay）是用小电流控制大电流的电子开关。常见于：
- 智能家居（灯光、家电控制）；
- 工业自动化（PLC）；
- 电力系统保护；
- 汽车电子；
- 安防系统；
- 远程控制；
- 电源切换；
- 启动大功率电机。

类型：
- **电磁继电器（EMR）**：机械触点、低成本；
- **固态继电器（SSR）**：无触点、长寿命；
- **磁保持继电器**：脉冲触发、保持状态、节能；
- **簧片继电器**：快速切换；
- **功率继电器**：高电流；
- **信号继电器**：低电平信号切换。


## 2. 工作原理与适用场景

### 2.1 电磁继电器

线圈通电产生磁场，吸引衔铁，带动触点动作。线圈失电时弹簧复位。

### 2.2 固态继电器

输入信号控制半导体开关（晶闸管/MOSFET），实现"无触点"开关。

### 2.3 磁保持继电器

脉冲触发驱动线圈改变状态，机械保持。不需要持续供电。

## 5. 硬件连接

- **续流二极管**：线圈反并 1N4148 或 1N4007；
- **驱动电流**：5V 继电器线圈 ~70 mA，MCU 直接驱动有限，需三极管（如 S8050）放大；
- **光耦隔离**：模块版通常带光耦，MCU 与继电器隔离；
- **触点保护**：感性负载（电机、线圈）加 RC 吸收或压敏电阻；
- **拉弧**：高压大电流时触点拉弧明显，影响寿命；
- **散热**：固态继电器大电流需散热片（> 30A）；
- **机械寿命**：电磁继电器寿命有限，频繁开关考虑 SSR。



感性负载（电机、变压器、线圈）断开时产生反向电动势，需要保护：

- **RC 吸收**：100Ω + 0.01µF 串联，并联在触点；
- **压敏电阻（MOV）**：电压限制；
- **续流二极管**：直流线圈用；
- **气体放电管**：高压保护。


## 6. 通信方式

- **GPIO**：直接控制（多数）；
- **I2C/SPI**：扩展芯片（PCF8574、MCP23017）；
- **RS485/Modbus**：工业总线控制；
- **WiFi/BLE**：智能继电器；
- **红外遥控**：智能家居。


## 7. 初始化流程

1. 配置 GPIO（输出模式）；
2. 设置初始为 OFF；
3. 上电稳定后开始控制；
4. 大电流需检查散热。


## 8. 驱动接口

```python
class Relay:
    def __init__(self, pin, active_high=True):
        self.pin = pin
        self.active_high = active_high

    def begin(self):
        self.off()

    def on(self):
        if self.active_high:
            self.pin.high()
        else:
            self.pin.low()

    def off(self):
        if self.active_high:
            self.pin.low()
        else:
            self.pin.high()

    def toggle(self):
        self.pin.toggle()


class SSRelay(Relay):
    """固态继电器。"""
    pass


class LatchingRelay:
    """磁保持继电器。"""
    def __init__(self, set_pin, reset_pin):
        self.set_pin = set_pin
        self.reset_pin = reset_pin

    def begin(self):
        pass

    def latch(self):
        """脉冲触发，闭合。"""
        self.set_pin.high()
        time.sleep_ms(50)
        self.set_pin.low()

    def unlatch(self):
        self.reset_pin.high()
        time.sleep_ms(50)
        self.reset_pin.low()
```


## 9. 数据格式与单位

- 控制信号：布尔（开/关）；
- 总线控制：寄存器地址 + 开关位；
- Modbus：线圈寄存器（功能码 0x01/0x05/0x0F）。


## 10. 错误处理

- **触点粘连**：过载或拉弧严重，更换继电器；
- **不响应**：线圈未通电、驱动电流不足；
- **触点氧化**：低电流负载易失效，使用"湿电路"或金触点型号；
- **SSR 失效**：过流或散热不足烧毁；
- **总线通信错误**：Modbus 地址错、波特率错。



- 高压电危险：220V AC 操作需专业；
- 触电防护：绝缘手套、工具；
- 火灾风险：触点粘连 + 大负载可能起火。


## 11. 低功耗

- 电磁继电器：线圈持续通电耗电；
- 磁保持继电器：脉冲后保持，零功耗；
- SSR：控制端 5~20 mA。


## 12. 示例代码

```python
relay = Relay(pin, active_high=True)
relay.begin()
relay.on()
time.sleep_ms(5000)
relay.off()

# 多路继电器
relays = [Relay(p1), Relay(p2), Relay(p3)]
for r in relays:
    r.begin()

# 切换
for i in range(2):
    relays[i].on()
    relays[(i+1) % 3].off()
    time.sleep_ms(1000)
```


## 13. 调试方法

1. 听继电器吸合声（咔嗒）；
2. 测触点两端电压（OFF 时 220V / ON 时 0V）；
3. 检查线圈电压；
4. 长时间运行测温；
5. 负载电流测量。


## 14. 常见问题

- **继电器不吸合**：线圈电压不足、驱动电流不够；
- **触点粘连**：过载或拉弧；
- **触点不导通**：触点氧化（低电流失效）；
- **SSR 不工作**：直流 SSR 接交流失败，或反之；
- **散热不足**：SSR 大电流烧毁；
- **线圈烧毁**：续流二极管未加，反向 EMF 击穿；
- **总线模块通信失败**：地址错、波特率不匹配。

## 16. 触点寿命

- 阻性负载：电气寿命长；
- 感性负载：拉弧严重，电气寿命下降；
- 容性负载（LED 灯、电容）：启动电流大，需浪涌保护；
- 电机负载：启动电流 5~10 倍，需降额使用。
