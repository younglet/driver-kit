# Current Monitoring


## 1. Module Overview

Current Monitoring measures the current consumption or current flow in a circuit. Common applications include:

- Battery capacity estimation (coulomb counting)
- Motor current monitoring
- Power measurement
- Over-current protection
- Solar cell monitoring
- Power supply debugging
- Fault diagnosis
- Factory energy management

Common implementation types:

- **Shunt resistor + op-amp**: classic solution
- **Hall-effect sensor**: isolated measurement
- **Dedicated current-monitor IC** (INA219 / INA226): high integration
- **Current transformer (CT)**: AC large-current measurement
- **Coulomb-counter IC**: battery-specific
- **Shunt (large current)**: high-current applications


## 2. Working Principles and Use Cases

### 2.1 Shunt resistor (low-side / high-side)

- A precision resistor of known value (milliohm range) is placed in series
- The voltage across the resistor is measured (mV range)
- `I = V / R`
- **Low-side measurement**: resistor between the load and GND (simple, but breaks GND reference)
- **High-side measurement**: resistor between the supply and the load (preserves shared GND)
- **Op-amp amplification**: amplifies the mV signal to the ADC range

### 2.2 Hall-effect sensor

- Uses the Hall effect to sense the magnetic field produced by the current
- Isolated measurement (no electrical contact)
- ACS712 outputs a voltage proportional to the current
- INA180 family integrates shunt + amplifier

### 2.3 Dedicated ICs

- INA219 / INA226 / INA228 integrate ADC + amplifier, output via I2C
- Measure voltage, current, and power simultaneously
- Accuracy up to ±0.1%

**Selection recommendations**:

- Small current (mA): shunt resistor + INA180
- Medium current (A): ACS712 / INA219 / INA226
- High current (10 ~ 100 A): ACS758 / current transformer + INA226
- High-side measurement: INA226 / INA228
- Battery state of charge: MAX17048 / CW2015 / BQ27441
- AC current: CT (e.g. ZMCT103F)
- Isolation required: Hall-effect or CT


## 5. Hardware Connection and Electrical Notes

### 5.1 Shunt resistor

- Low-side (simple but breaks GND reference)
- High-side (preserves shared GND)
- Resistor accuracy: ±1% or better
- Low temperature coefficient (TCR < 50 ppm/°C)
- 4-wire (Kelvin) connection reduces lead-resistance error
- Power dissipation: `P = I² × R` (e.g. 10 A × 10 mΩ = 1 W)
- Provide thermal relief

### 5.2 Op-amp amplification

- Gain: 100 ~ 200 is suitable
- Offset voltage < 50 µV
- Small bias current
- Common-mode rejection ratio > 100 dB

### 5.3 Hall-effect sensor

- Module powered from 3.3 V / 5 V
- Output voltage proportional to current
- Midpoint voltage = `VCC / 2` (bidirectional)
- Mind the common-mode voltage range
- Module supply voltage must not exceed the rating

### 5.4 Filtering

- Low-pass RC filter (bandwidth limit)
- Software median / averaging filter
- Removes switching-supply noise


## 6. Communication Method or Control Signal

- Analog: directly into ADC
- I2C: INA219 / INA226 / MAX17048
- SPI: INA238 / ADS131
- Digital: some chips have a built-in ADC outputting digital values

## 8. Common Driver Interfaces

```python
class CurrentSensor:
    """Shunt resistor + instrumentation amplifier + ADC."""
    def __init__(self, adc, shunt_ohm, gain=100, vref=3.3):
        self.adc = adc
        self.shunt = shunt_ohm
        self.gain = gain
        self.vref = vref

    def read_current(self):
        """Return current in amperes."""
        v = self.adc.read_voltage()
        # v = i × shunt × gain
        i = v / (self.shunt * self.gain)
        return i


class ACS712:
    """ACS712 Hall-effect current sensor."""
    def __init__(self, adc, sensitivity_mv_a=185, vcc=5.0):
        self.adc = adc
        self.sens = sensitivity_mv_a / 1000  # V/A
        self.zero = vcc / 2  # midpoint voltage

    def read_current(self):
        v = self.adc.read_voltage()
        return (v - self.zero) / self.sens
```

A coulomb counter integrates current to estimate the remaining capacity:

```python
class CoulombCounter:
    def __init__(self, current_sensor, dt_seconds=1):
        self.cs = current_sensor
        self.dt = dt_seconds
        self.accumulated = 0  # mAh

    def update(self):
        i_mA = self.cs.read_current() * 1000  # mA
        # mAh = mA × hours
        self.accumulated += i_mA * self.dt / 3600

    def remaining_mah(self, initial_mah):
        return initial_mah - self.accumulated

    def percentage(self, initial_mah):
        return max(0, self.remaining_mah(initial_mah) / initial_mah * 100)
```

## 10. Calibration, Compensation, and Filtering

### 10.1 Shunt resistor calibration

- Measure the actual resistance with a multimeter
- Record the precise value for software calculation

### 10.2 Zero-point calibration

- Read 0 when no current flows
- Compute the zero-point offset

### 10.3 Gain calibration

- Calibrate with a known current source
- Adjust the scaling factor

### 10.4 Temperature compensation

- Resistor TCR affects accuracy
- Real-time temperature compensation (some ICs include this)

Other filters:

- Median filter: removes spikes
- Moving average: low-pass behavior
- α-β filter: state estimation
- Kalman filter: optimal estimation

## 12. Error Handling and Exception Recovery

- **Jittery readings**: insufficient filtering, poor shunt contact
- **Zero-point drift**: temperature drift
- **Range exceeded**: current out of range
- **Burned out by overload**: current above shunt power rating
- **EMI disturbance**: switching-supply noise


## 13. Low Power

- ADC low-power mode
- Intermittent measurement
- Measure on wake-up (off while sleeping)
- INA226 standby < 1 µA

## 15. Debugging Methods

1. Measure the voltage across the shunt with a multimeter (verification)
2. Use an oscilloscope to view the waveform (switching-supply ripple)
3. Verify with an ammeter in series
4. Measure the common-mode voltage
5. Check the zero-point offset
6. Test over-current protection


## 16. Frequently Asked Questions

- **Inaccurate current reading**: shunt tolerance, op-amp offset, thermal drift
- **Jittery readings**: switching-supply noise, insufficient filtering
- **Zero-point drift**: temperature changes, recalibrate
- **Insufficient range**: resistor too small or too large
- **Distorted measurement**: common-mode voltage exceeded
- **Burned out**: overload or reverse polarity
