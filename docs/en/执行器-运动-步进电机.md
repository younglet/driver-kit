# Stepper Motor


## 1. Module Overview

A stepper motor converts electrical pulses into precise angular displacement. High-precision position control can be achieved even with open-loop control. Common uses:

- 3D printers
- CNC engraving machines
- Pen plotters / drawing machines
- Camera gimbals
- Automation equipment
- Valve control
- Pick-and-place machines
- Lab automation

Types:

- **Permanent Magnet (PM)**: permanent-magnet rotor, large step angle
- **Variable Reluctance (VR)**: non-magnetic rotor, low torque
- **Hybrid (HB)**: combines PM and VR, most common
- **Unipolar**: 5/6 wires with center-tapped windings
- **Bipolar**: 4 wires, full-bridge drive


## 2. Working Principle and Use Cases

Stator coils are energized in a specific sequence to attract the rotor teeth into alignment. Changing the energizing sequence rotates the rotor by one step angle.

- **Full step**: one phase (unipolar) or two phases (bipolar) per step
- **Half step**: alternate full-step modes; step angle is halved
- **Microstep**: subdivided current with sinusoidal control; finer step (1/16, 1/32, 1/256)

Common step angles: 1.8° (200 steps/rev), 0.9° (400 steps/rev).

## 4. Key Parameters and Selection Guide

| Parameter | Description |
|---|---|
| Step angle | 1.8° / 0.9° / 0.36° etc. |
| Holding torque | N·m (important) |
| Phase current | A / phase |
| Phase resistance | Ω |
| Phase inductance | mH |
| Rotor inertia | g·cm² |
| Pole pairs | - |
| Wire count | 4 (bipolar) / 5-6 (unipolar) |
| Working voltage | typically 12V / 24V / 48V |
| Operating temperature | 80°C max |
| Lifetime | long (bearing wear) |

**Selection suggestions**:

- Beginner / maker: 28BYJ-48, NEMA17
- 3D printer: NEMA17 (0.4~1.5 N·m)
- Small CNC: NEMA17 + closed-loop driver
- Large CNC: NEMA23 / NEMA34
- No lost steps: closed-loop stepper + encoder


## 5. Hardware Connection

- Driver: must use a stepper driver (see Motor Driver doc)
  - ULN2003 (unipolar 28BYJ-48)
  - A4988 / DRV8825 / TMC2208 / TMC2209 (bipolar NEMA17)
- Wiring: 4 wires for bipolar (A+/A-/B+/B-)
- Power supply: constant-current drive (built into driver)
- Heat dissipation: driver needs a heatsink for high-current operation
- Signal lines: STEP (pulse) + DIR (direction) + EN (enable)
- Microstep setting: MS1/MS2/MS3 jumpers (TMC uses UART/SPI)
- Anti-miswiring: A+/A- swapped will cause whining but no rotation


## 6. Communication Method

- **STEP + DIR**: pulse + direction (most common)
- **CW/CCW**: dual-pulse forward/reverse
- **SPI**: TMC series configuration
- **UART**: TMC2209 single-wire configuration


## 7. Initialization Flow

1. Wire the driver + motor
2. Set microstep (MS jumpers)
3. Set current (Vref adjustment)
4. Enable motor (EN pin low)
5. Send pulses to start motion
6. Configure acceleration/deceleration profile


## 8. Driver Interface

```python
class Stepper:
    def __init__(self, step_pin, dir_pin, en_pin=None):
        self.step = step_pin
        self.dir = dir_pin
        self.en = en_pin
        self.steps_per_rev = 200  # 1.8°

    def begin(self):
        if self.en:
            self.en.low()  # enable
        self.dir.low()

    def move_steps(self, steps, delay_us=1000):
        """Pulse the motor for the specified number of steps."""
        for _ in range(steps):
            self.step.high()
            time.sleep_us(delay_us)
            self.step.low()
            time.sleep_us(delay_us)

    def set_speed_rpm(self, rpm):
        """Set RPM via step delay."""
        step_delay_us = 60_000_000 // (self.steps_per_rev * rpm)
        self.step_delay = step_delay_us

    def rotate(self, degrees, direction=1):
        """Rotate by the specified angle."""
        steps = int(self.steps_per_rev * degrees / 360)
        if direction < 0:
            self.dir.high()
        else:
            self.dir.low()
        self.move_steps(steps)
```



Too-fast start/stop will lose steps; an S-curve or trapezoidal acceleration is required:

```python
def trapezoidal_profile(distance, v_max, a):
    """Returns (t, position) sequence."""
    t_accel = v_max / a
    d_accel = 0.5 * a * t_accel ** 2
    d_decel = d_accel
    d_const = distance - d_accel - d_decel
    if d_const < 0:
        # Triangle profile
        t_accel = (distance / a) ** 0.5
        t_decel = t_accel
        t_const = 0
        # ...
    else:
        t_const = d_const / v_max
    return t_accel, t_const, t_decel
```



- Full step: full current per step
- Half step: 1/2 current on adjacent phases
- Microstep: 1/4, 1/8, 1/16, 1/32, 1/256 current ratios
- TMC StealthChop quiet drive: smooth sinusoidal microstepping


## 9. Data Format, Units, and Timing

- Pulse: digital signal (rising edge triggers)
- Direction: high/low level
- Speed: RPM or PPS (Pulse Per Second)
- Acceleration: rad/s² or steps/s²
- Current: A


## 10. Error Handling and Exception Recovery

- **Lost steps**: acceleration too fast, insufficient torque, current too low
- **Whining**: current too high, A/B phases swapped
- **Overheating**: current too high, insufficient cooling
- **Vibration**: low-speed resonance, avoid resonance points
- **Driver alarm**: over-current, over-temperature, short circuit


## 11. Low Power

- Disable current when stopped (EN pin or standby mode)
- TMC StealthChop has low quiescent current
- Reduce holding torque to save energy


## 12. Example Code

```python
stepper = Stepper(step_pin, dir_pin, en_pin)
stepper.steps_per_rev = 200
stepper.begin()

stepper.set_speed_rpm(60)
stepper.rotate(360, direction=1)  # forward 360°
time.sleep_ms(500)
stepper.rotate(-360, direction=-1)  # reverse 360°
```


## 13. Debugging Methods

1. Slowly turn and observe the step interval
2. Test microstep smoothness
3. Check holding torque (holds position when de-energized)
4. Test acceleration/deceleration (max speed)
5. Check heating (long run < 70°C)
6. Adjust current (Vref)


## 14. Frequently Asked Questions

- **Motor does not turn**: wiring wrong, no signal, driver not enabled
- **Only vibrates**: A/B phase order wrong
- **Lost steps**: acceleration too fast, insufficient torque, load too high
- **Severe heating**: current set too high
- **High-speed lost steps**: increase supply voltage (e.g., 24V → 36V), increase current
- **Resonance noise**: avoid low-speed resonance points, or use microstepping

## 16. Closed-Loop Control

Closed-loop stepper (with encoder):

- No lost steps (even at high-load acceleration)
- TMC closed loop
- Encoder feedback for position
- Higher cost than open loop
