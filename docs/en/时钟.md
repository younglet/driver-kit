# Clock Overview

A Real-Time Clock (RTC) is the time reference for embedded systems. DriverKit covers **1 clock module**.

## Module Quick Index

| Module | Description | Doc |
|---|---|---|
| RTC | 32.768 kHz + backup battery | [Open](./时钟-实时时钟-RTC) |

## Selection Quick Reference

| Application | Recommended |
|---|---|
| Beginner / maker | DS1307 (depends on external crystal, accuracy ±2 ppm) |
| High precision / perpetual calendar | DS3231 (built-in TCXO, ±2 ppm ≈ ±1 min/year) |
| Ultra-low power | PCF8563 / RX8900 |
| Domestic alternative | SD2405 / SD3088 |
| Network time sync | NTP (Wi-Fi / Ethernet) |
| Satellite time service | GNSS PPS |

## Common Knowledge

- **CR2032 backup battery** provides retention on power loss (5~10 years)
- **32.768 kHz crystal** is the time base
- **BCD format** stores time (0x25 = 25 seconds)
- **Leap year compensation** is handled automatically by most RTCs
- **Alarm interrupt** can wake up the MCU
