# Sensors Overview

Sensors are the eyes and ears of an embedded system. DriverKit documents **17 sensor modules**, organized into **five fields**: Orientation, Environment, Distance, Proximity, and Vision.

## Common Knowledge

- **I2C dominates** — most modern sensors use it.
- **Calibration is mandatory**: accelerometers / gyroscopes need zero-bias calibration; magnetometers need hard-iron / soft-iron compensation.
- **Coordinate axes follow each datasheet** — orientation differs between vendors.
- **Temperature drift affects accuracy** — high-precision applications need temperature compensation.
