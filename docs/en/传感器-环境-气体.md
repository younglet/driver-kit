# Gas Sensor


## 1. Module Overview

A gas sensor detects the concentration of specific gases in air, such as combustible gases (CH₄, propane, LPG), toxic gases (CO, H₂S, NH₃), and air-quality indicators (VOC, CO₂, PM2.5).

Types:
- **Semiconductor (MOS)**: MQ series, SGP series
- **Electrochemical**: industrial-grade CO, H₂S, NO₂
- **NDIR (Non-Dispersive Infrared)**: high-precision CO₂, CH₄
- **PID (Photoionization)**: high-precision VOC
- **Laser scattering**: PM2.5
- **Catalytic combustion**: combustible gases


## 2. Working Principle and Use Cases

### 2.1 Semiconductor (MOS)

A sensing material (SnO₂, etc.) changes resistance after adsorbing target gas. A heater (about 5V / 200 mW) keeps the surface activated.

Use cases: combustible-gas alarms, indoor air quality, residential gas leak detection.

### 2.2 Electrochemical

The gas diffuses through a membrane to the working electrode and undergoes redox reactions in the electrolyte, producing a current signal.

Use cases: industrial safety (CO, H₂S, O₂, NO₂), medical, environmental monitoring.

### 2.3 NDIR

An infrared source passes through the gas; the detector measures the attenuation at characteristic wavelengths.

Use cases: CO₂ monitoring (HVAC, automotive, greenhouses), CH₄.

### 2.4 PM2.5 Laser Scattering

Laser light is scattered by suspended particles; the scattering intensity correlates with concentration.

Use cases: air-quality monitoring, purifiers, outdoor dust.

| Parameter | Description |

| Range | ppm / ppb / mg/m³ |

| Accuracy | ±5 ppm / ±50 ppm |

| Response time T90 | Seconds to minutes |
| Recovery time | Seconds to minutes |
| Operating temperature | Wide for semiconductor; -20~+50°C for electrochemical |
| Operating humidity | 15~90% RH (non-condensing) |
| Lifetime | Semiconductor 5~10 yr; electrochemical 2~3 yr; NDIR 10+ yr |
| Warm-up time | Semiconductor 30~120 s; electrochemical several minutes |
| Long-term drift | Significant for semiconductor; also present in electrochemical |
| Interface | Analog / I2C / SPI / UART / PWM |
| Power | 3.3V / 5V / 24V |

**Selection suggestions**:
- Home gas leak alarm: MQ-2, MQ-5, MQ-9
- Air quality (VOC): SGP40, SGP41, ENS160, BME680
- CO₂ monitoring: SCD30, SCD40, MH-Z19
- PM2.5: PMS5003, PMSA003, SPS30
- Industrial gas safety: electrochemical (e.g. CO, H₂S, O₂ probes)
- Formaldehyde: ZE08-CH2O, electrochemical formaldehyde sensors


## 5. Hardware Connection

- **Semiconductor (MQ series)**: 5V heater, ~200 mW; AO analog output + DO digital output (adjustable threshold)
- **Electrochemical**: op-amp circuit (transimpedance amplifier, TIA) amplifies nA/pA currents to mV; watch for polarization and reverse currents
- **NDIR**: 5V / 12V / 24V supply, IR source requires driver
- **Laser scattering**: fan / pump driver
- Keep away from: high temperature, moisture, oil, dust
- Ventilation: ensure the target gas reaches the sensor


## 6. Communication Method

- **Analog output**: AO pin, ADC reads voltage
- **Digital output**: threshold comparison (DO pin)
- **I2C**: SGP40, SCD40, ENS160, CCS811, BME680
- **SPI**: some electrochemical chips
- **UART**: MH-Z19, PMS5003 (active report mode)
- **PWM**: NDIR sensors like SCD30


## 7. Initialization Flow

1. Power up
2. Wait for warm-up (semiconductor 30~120 s; electrochemical ~5 min)
3. Read calibration coefficients (e.g. SCD40)
4. Set operating mode (single / continuous / periodic)
5. Set sample rate
6. Trigger or read measurement

## 9. Data Format and Units

- ppm: parts per million
- ppb: parts per billion
- mg/m³: mass concentration (PM2.5)
- VOC index: SGP40 outputs 0~500
- eCO₂: equivalent CO₂ (algorithm-estimated, not real NDIR)
- Analog output: voltage (V) → concentration requires lookup or fitting


## 10. Calibration and Filtering

### 10.1 Semiconductor

- Limited factory calibration
- Build a voltage-concentration curve using a known-concentration gas
- Recalibrate after long-term use

### 10.2 NDIR

- Single-point zero calibration (in N₂ or via the automatic ABC logic in fresh air)
- SCD40 supports Automatic Self-Calibration (ASC)
- Two-point calibration gives higher accuracy (requires reference gas)

### 10.3 PM2.5 Laser

- Factory-calibrated; validate with reference particulate sources (e.g. cigarette smoke)
- Humidity affects particle size — apply humidity compensation


## 11. Interrupts and DMA

- Data-ready interrupt
- Threshold alarm (CO exceeds limit)
- Most gas sensors have low data rates (once per minute) — FIFO is not required


## 12. Error Handling

- Heater failure: MQ series heater wire broken
- Analog output flat: sensor saturated or failed
- I2C error: check address and pull-ups
- Long-term drift: recalibrate periodically
- Poisoning: electrochemical sensor exposed to high concentration loses accuracy and must be replaced


## 13. Low Power

- Semiconductor: heater can be turned off to reduce power (at the cost of response)
- SGP40: measurement 34 mA, standby 0.5 mA
- Electrochemical: measurement mA, standby µA
- Intermittent measurement: 1 sample / minute is enough for indoor monitoring

## 15. Debugging Methods

1. Compare CO₂ between a ventilated room and a closed room
2. Observe PM2.5 outdoors
3. Breathe on the VOC sensor and watch the reading change
4. Power the heater (MQ series) and observe the current
5. Calibrate with reference gas when high accuracy is required


## 16. Frequently Asked Questions

- **MQ series baseline unstable**: humidity and temperature affect it; temperature compensation is required
- **Electrochemical CO drifts**: drift is large; recalibrate periodically
- **NDIR long-term offset**: ABC logic auto-calibrates; ensure the indoor environment has some ventilation periods
- **SGP40 VOC index 0**: not connected or warm-up not complete
- **PM2.5 reading high**: humidity inflates particle size; apply humidity compensation
- **Laser sensor dirty**: fan failure or dust clog

## 18. Common Algorithms

### 11.1 VOC Index

Sensirion SGP40 outputs a 0~500 index (1 = clean, 500 = extremely poor).

### 11.2 eCO₂

Equivalent CO₂ estimated from VOC (CCS811, ENS160).

### 11.3 ABC Logic

NDIR CO₂ sensors apply Automatic Baseline Correction: when a low concentration (< 400 ppm) is observed for a long period, the sensor assumes 400 ppm and calibrates itself.
