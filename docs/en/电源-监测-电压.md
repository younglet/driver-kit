# Voltage Monitoring


## 1. Module Overview

Voltage Monitoring is the most commonly used power-sensing function in embedded systems. Typical uses include:

- Battery voltage monitoring
- Power failure detection
- Under-voltage protection
- Energy harvesting
- USB Type-C voltage detection
- Solar charging monitoring
- Multi-rail power switching
- ADC measurement

Common implementation types:

- **Resistor divider + MCU ADC**: lowest cost
- **Dedicated ADC chip**: high accuracy
- **Dedicated power-monitor chip**: with reset and interrupt
- **Current/power monitor IC** (e.g. INA219): multi-parameter
- **Voltage reference IC**: precision source
- **Digital voltmeter module**: ready-to-use module


## 2. Working Principles and Use Cases

### 2.1 Resistor divider

- High voltage → resistor divider → within MCU ADC range
- `Vadc = Vin × R2 / (R1 + R2)`
- Choose `R1`/`R2` ratio so that the maximum voltage maps to the ADC full-scale value
- Add an RC filter (to suppress noise)
- Add a Zener diode (for voltage clamp protection)

### 2.2 ADC resolution

- 8-bit (0.4% / 256 steps)
- 10-bit (0.1% / 1024 steps)
- 12-bit (0.024% / 4096 steps)
- 16-bit (0.0015% / 65536 steps)
- 24-bit (high-precision measurement)

### 2.3 Voltage reference

- **MCU internal reference**: 1.1V / 2.5V / 3.0V (low accuracy, ±5%)
- **External reference**: TL431, REF5025, ADR01/02/05
- **Built-in reference of high-precision ADC**: e.g. ADS1220 (2.048V internal reference)
- The reference choice directly affects measurement accuracy.

## 5. Hardware Connection and Electrical Notes

### 5.1 Resistor divider

- `R1` on the high-voltage side, `R2` on the low-voltage side
- Divider ratio = `R2 / (R1 + R2)`
- `R1 + R2` is typically 10 kΩ ~ 1 MΩ (larger → less current, but more noise)
- Use 0.1%-precision resistors for high accuracy (at higher cost)
- ADC input impedance must be much larger than `R2` (to avoid divider error)

### 5.2 Filtering

- 100 nF + 10 kΩ RC low-pass
- Cutoff frequency ~ 1.6 kHz
- Filters out high-frequency noise

### 5.3 Protection

- TVS diode (against transient voltage)
- Series resistor (current limit)
- Zener diode (voltage clamp)
- Input capacitor (filter spikes)

### 5.4 High-voltage isolation

- Isolation amplifier (e.g. ACPL-C870)
- Voltage transformer (ZMPT101B)
- Differential amplifier


## 6. Communication Method or Control Signal

- Analog: resistor divider directly into ADC
- I2C: ADS1115, INA219, INA226
- SPI: MCP3008, ADS1220, ADS1256
- Interrupt: threshold interrupt (e.g. low-voltage alert)

## 8. Common Driver Interfaces

```python
class VoltageSensor:
    """Resistor divider + MCU ADC."""
    def __init__(self, adc, r1, r2):
        self.adc = adc
        self.r1 = r1
        self.r2 = r2

    def read_voltage(self):
        """Return the measured voltage in volts."""
        v_adc = self.adc.read_voltage()
        return v_adc * (self.r1 + self.r2) / self.r2

class BatteryMonitor:
    """Battery monitor (divider + ADC + coulomb counter)."""
    def __init__(self, adc):
        self.adc = adc

    def voltage(self):
        """Return battery voltage in volts."""
        return self.adc.read_voltage() * self.scale

    def percentage(self):
        """Return state-of-charge percentage (voltage-based)."""
        v = self.voltage()
        if v > 4.2:
            return 100
        elif v < 3.0:
            return 0
        else:
            return (v - 3.0) / (4.2 - 3.0) * 100

    def low_battery(self):
        """Return True if battery is low."""
        return self.voltage() < 3.3
```

Typical Li-Ion discharge curve:

- 4.2V: 100% (fully charged)
- 3.85V: ~50%
- 3.6V: ~20%
- 3.3V: ~5%
- 3.0V: 0% (recommended end-of-discharge)

The exact relationship depends on load and cell characteristics; please refer to the cell vendor's curve.

## 10. Calibration, Compensation, and Filtering

### 10.1 Gain calibration

- Calibrate against a known voltage source (e.g. 3.000 V reference)
- Compute the actual divider ratio
- Apply a correction factor

### 10.2 Offset calibration

- Read 0 with the input grounded
- Compute and store the offset

### 10.3 Calibration flow

```python
def calibrate(adc, ref_voltage):
    raw = adc.read_raw()
    v = adc.read_voltage()
    return ref_voltage / v  # gain correction factor
```

Other filters:

- Median filter (to remove spikes)
- Moving average (10 ~ 100 samples)
- Weighted moving average (higher weight on new samples)
- Low-pass filter (software RC)


## 11. Interrupts, DMA, FIFO, and Buffers

- Under-voltage interrupt (battery low-voltage protection)
- Over-voltage interrupt (over-voltage protection)
- Threshold interrupt (resistor divider + comparator)
- ADC conversion-complete interrupt


## 12. Error Handling and Exception Recovery

- **ADC error**: I2C / SPI communication failure, reading 0xFFFF
- **Calibration failure**: inaccurate reference voltage
- **Divider failure**: cold solder joints, damaged resistor
- **Input over-range**: clamp diode did not engage
- **EMI disturbance**: add filtering, optimize PCB routing


## 13. Low Power

- ADC low-power mode (µA level)
- Intermittent measurement (once per minute is enough for battery monitoring)
- Disable the ADC when not in use
- Threshold interrupt to wake the MCU

## 15. Debugging Methods

1. Measure the voltage across each divider resistor
2. Measure the ADC input voltage
3. Cross-check with a multimeter
4. Measure the supply ripple
5. Verify calibration (reference voltage)
6. Test the under-voltage protection


## 16. Frequently Asked Questions

- **Inaccurate reading**: divider resistor tolerance, ADC accuracy, reference voltage drift
- **Low-voltage alert not firing**: threshold misconfigured, interrupt not enabled
- **Large ripple**: unstable supply, add filtering
- **Reading jumps**: EMI, insufficient filtering
- **ADC burned out**: input over voltage
- **Inaccurate battery percentage**: load current affects voltage, consider a coulomb counter


## 17. References

- The ADC input impedance must be much larger than `R2`
- High voltage must be divided before reaching the MCU
- Add protection circuitry (TVS, Zener)
- Use a precision reference for calibration
- The battery percentage is only an estimate (depends on current and temperature)
- Industrial applications require isolation amplifiers
