# Water Pump


## 1. Module Overview

A water pump transports liquid via a motor driver. Common types:

- **DC diaphragm pump**: low flow, high pressure
- **Centrifugal pump**: high flow, low pressure
- **Peristaltic pump**: precise metering, no contamination
- **Gear pump**: precise flow, high pressure
- **Piston pump**: high-pressure pulsation
- **Submersible pump**: works underwater
- **Micro vacuum pump**: pumps air / water

Applications:

- Water cooling systems
- Automatic irrigation
- Fountains and landscaping
- Lab instruments
- Medical (dialysis, infusion)
- Chemical liquid transfer
- Coffee machines
- 3D printers (coolant circulation)
- Automotive cooling


## 2. Working Principle and Use Cases

### 2.1 Diaphragm Pump

Motor drives an eccentric cam → diaphragm reciprocates → check valves alternate open/close → liquid flows in one direction.

### 2.2 Centrifugal Pump

Motor drives the impeller at high speed → centrifugal force throws liquid out → water flow forms.

### 2.3 Peristaltic Pump

Motor drives rollers → compresses tubing → pushes liquid unidirectionally; the liquid only contacts the tubing inner wall.

### 2.4 Gear Pump

Two gears mesh and rotate → liquid is pushed between the gear teeth.

**Selection suggestions**:

- Beginner / water cooling: 370 diaphragm, 385 diaphragm
- High flow: 880 diaphragm, centrifugal pump
- Precise metering: peristaltic pump (Kamoer, LongerPump)
- Vacuum: micro vacuum pump
- High pressure: gear pump, piston pump
- Outdoor / underwater: submersible pump


## 5. Hardware Connection

- **Drive current**: 12V diaphragm pump 1~3A, needs MOSFET or dedicated driver
- **Flyback diode**: reverse-parallel across motor coil
- **Decoupling capacitors**: 100nF + 100µF
- **PWM speed control**: supported by some pumps (1~10 kHz)
- **H-bridge direction control**: peristaltic pump can reverse (suck / pump)
- **Inlet hose**: avoid kinks and air leaks
- **Outlet hose**: watch back pressure
- **Waterproofing**: submersible pump IP68
- **Heat dissipation**: high-power pumps need cooling for long operation
- **Filtration**: add a filter screen at the inlet


## 6. Communication Method

- **GPIO / MOSFET**: basic on/off control
- **PWM**: speed control (most pumps support)
- **H-bridge + PWM**: peristaltic pump forward/reverse
- **UART / RS485**: smart pump control (e.g., Kamoer KPP)
- **Analog voltage**: some industrial pumps


## 7. Initialization Flow

1. Wiring (power, control signals)
2. Check inlet hose sealing
3. Prime with liquid (before first use)
4. Test run (low speed → high speed)
5. Calibrate flow (peristaltic pump)


## 8. Driver Interface

```python
class Pump:
    def __init__(self, pin):
        self.pin = pin

    def begin(self):
        pass

    def on(self):
        self.pin.high()

    def off(self):
        self.pin.low()


class PwmPump:
    """Variable-speed pump."""
    def __init__(self, pwm):
        self.pwm = pwm

    def begin(self):
        self.pwm.duty(0)

    def set_speed(self, percent):
        """0~100."""
        self.pwm.duty(percent)


class PeristalticPump:
    """Peristaltic pump (bidirectional)."""
    def __init__(self, in1, in2, pwm):
        self.in1 = in1
        self.in2 = in2
        self.pwm = pwm

    def begin(self):
        self.stop()

    def forward(self, speed):
        self.in1.high()
        self.in2.low()
        self.pwm.duty(speed)

    def backward(self, speed):
        self.in1.low()
        self.in2.high()
        self.pwm.duty(speed)

    def stop(self):
        self.in1.low()
        self.in2.low()
        self.pwm.duty(0)

    def dispense_ml(self, ml, rate_ml_min=10):
        """Dispense a fixed volume based on flow and time."""
        seconds = ml * 60 / rate_ml_min
        self.forward(50)
        time.sleep(seconds)
        self.stop()
```


## 9. Data Format, Units, and Timing

- PWM duty cycle: 0~100%
- Flow: L/min, mL/min
- Pressure: MPa, bar, kPa
- Metered dispensing: mL, mL/min
- Status: voltage, current, temperature


## 10. Calibration, Compensation, and Filtering

- Peristaltic pump: measure actual flow to calibrate (mL/pulse or mL/s)
- Diaphragm pump: flow-speed characteristic curve
- Centrifugal pump: head-flow (H-Q) curve
- Store calibration data with version number + CRC


## 11. Error Handling and Exception Recovery

- **No water intake**: inlet air leak, insufficient self-priming
- **Insufficient flow**: pump clogging, excessive back pressure
- **Overheating**: long-time heavy load, blockage
- **Loud noise**: cavitation, blockage
- **Motor burnt out**: stall, overload
- **Leak**: seal aging



- High-pressure liquid is dangerous
- Chemical media require corrosion-resistant pumps
- Food / medical applications need food-grade certification
- Outdoor use: pay attention to waterproofing and lightning protection


## 12. Low Power

- Intermittent operation
- PWM speed control (slow down when flow demand is low)
- Power off when idle

## 14. Debugging Methods

1. Test no-load operation
2. Test loaded flow
3. Measure head (static vs actual)
4. Measure current (load variation)
5. Measure temperature (long-time operation)
6. Check sealing (no bubbles, no leak)


## 15. Frequently Asked Questions

- **No water intake**: inlet air leak or no water
- **Insufficient flow**: filter clogging, motor power insufficient
- **Loud noise**: cavitation (insufficient inlet water)
- **Pump overheating**: long-time heavy load
- **Motor burnt out**: stall for too long
- **Peristaltic pump accuracy drops**: tubing aging, flow calibration invalid
