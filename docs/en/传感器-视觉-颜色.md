# Color Sensor


## 1. Module Overview

A color sensor uses spectral splitting or photodiodes to detect RGB or finer spectral components. It is used for:
- Color identification and matching
- LED binning
- Industrial color inspection
- Automatic color-temperature measurement
- Lighting color calibration
- Printing / textile inspection
- Education / maker projects

Types:
- **RGB color sensor**: TCS34725, APDS-9960, TCS3200
- **Spectral sensor**: AS7341 (11 channels), AS7265x
- **Color temperature sensor**: TCS3430
- **XYZ tristimulus**: high-end industrial
- **NIR / multi-spectral**: agriculture, food inspection


## 2. Working Principle and Use Cases

### 2.1 RGB Color

Photodiodes + RGB filter array measure red / green / blue intensity separately. Tristimulus values X / Y / Z can be derived from RGB.

### 2.2 Multi-channel Spectral

Multiple independent photodiodes with different wavelength filters (8~18 channels) output the intensity in each band — ideal for fine-grained measurement.

### 2.3 White Balance / Color Temperature

Compute color temperature and illuminance from the four RGBC (red, green, blue, clear) channels.

## 4. Key Parameters and Selection Guide

| Parameter | Description |
|---|---|
| Channel count | 3 (RGB) / 4 (RGBC) / 6~18 (spectral) |
| Filter | IR blocking (affects accuracy) |
| Dynamic range | Above 1:10000 |
| Resolution | 16-bit |
| Sensitivity | Adjustable gain |
| Integration time | 1.6 ms ~ 700 ms |
| Interface | I2C |
| Interrupt | Threshold interrupt |
| Supply voltage | 2.7V~3.6V |
| Power consumption | Measurement ~0.5 mA, standby µA-level |

**Selection suggestions**:
- Entry: TCS3200 (frequency output, simple)
- RGB color identification: TCS34725, APDS-9960
- Color temperature / white balance: TCS3430, BH1749
- Multi-spectral (professional): AS7341, AS7265x
- Proximity + gesture: APDS-9960
- UV detection: VEML6075


## 5. Hardware Connection

- I2C pull-ups: 4.7kΩ
- INT: interrupt output (threshold)
- LED pin: TCS34725 etc. include an LED driver
- Lens: photodiodes need a lens to focus
- IR blocking: TCS34725 includes an IR-blocking filter
- Proximity: APDS-9960 IR LED + detector
- Watch the light intensity: under strong light, reduce gain or shorten integration time


## 6. Communication Method

- I2C: standard 100 / 400 kHz
- Frequency output: TCS3200 (output square-wave frequency proportional to color components)
- Some support SMBus Alert


## 7. Initialization Flow

1. Configure I2C
2. Read CHIP_ID
3. Soft-reset
4. Set integration time
5. Set gain (1x / 4x / 16x / 60x)
6. Start measurement
7. Wait for interrupt or poll
8. Read RGB / Clear data

## 9. Data Format and Units

- RGB: 16-bit counts per R/G/B channel
- Clear: white-light intensity
- Lux: illuminance
- CCT: color temperature (K)
- CIE 1931 (x, y): chromaticity coordinates
- HSI: hue / saturation / intensity


## 10. Calibration and Filtering

- White-point calibration: read RGB under a known color-temperature light to determine the white-point offset
- Matrix calibration: use a color chart (e.g. X-Rite ColorChecker) for multi-color calibration
- Dark compensation: black-balance correction
- Distance normalization: scattering attenuation differs with distance to the target


## 11. Interrupts and DMA

- Threshold interrupt: Clear channel exceeds threshold
- Some chips support FIFO (multi-sample buffer)


## 12. Error Handling

- I2C error: check address and pull-ups
- Reading saturated: adjust gain or integration time
- Wrong color judgement: white balance not calibrated
- Ambient light interference: re-calibrate periodically under changing light


## 13. Low Power

- Single measurement then sleep
- Standby < 1 µA
- Intermittent measurement: 1 Hz is enough

## 15. Debugging Methods

1. Test different colored objects (red, green, blue, white, black)
2. Test different light sources (daylight, incandescent, LED)
3. Test color-temperature estimation
4. Cross-check against a color chart
5. Test ambient light interference


## 16. Frequently Asked Questions

- **Color judgement inaccurate**: white balance not calibrated
- **Saturated**: strong light — adjust gain
- **TCS3200 frequency method has large error**: use an integrated IC (TCS34725)
- **Proximity not responding**: APDS-9960 LED not enabled
- **IR interference**: choose a chip with built-in IR blocking filter

## 18. RGB → HSI / HSV Conversion

```python
def rgb_to_hsv(r, g, b):
    r, g, b = r/255.0, g/255.0, b/255.0
    mx, mn = max(r, g, b), min(r, g, b)
    df = mx - mn
    if mx == r:
        h = ((g - b) / df) % 6
    elif mx == g:
        h = (b - r) / df + 2
    else:
        h = (r - g) / df + 4
    h *= 60
    s = 0 if mx == 0 else df / mx
    v = mx
    return h, s, v
```
