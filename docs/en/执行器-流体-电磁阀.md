# Solenoid Valve


## 1. Module Overview

A solenoid valve is a fluid valve controlled by an electromagnet. Common uses:

- Pneumatic control systems (robot arms, cylinders)
- Hydraulic control
- Water flow control (washing machines, dishwashers, water purifiers)
- Fuel control
- Medical equipment (ventilators, anesthesia machines)
- Automatic irrigation
- Coffee machines
- Industrial automation

Types:

- **Direct-acting**: solenoid force directly opens/closes the valve core
- **Pilot-operated**: solenoid opens a pilot orifice; main valve uses pressure differential
- **Proportional valve**: continuously controls flow / pressure
- **2-way (2/2)**: single passage
- **3-way (3/2)**: single passage + exhaust
- **5-way (5/2)**: dual passages + exhaust
- **Pneumatic, hydraulic, water** valves


## 2. Working Principle and Use Cases

When the coil is energized, it produces a magnetic field that pushes the valve core:

- Direct-acting: solenoid force > fluid pressure → opens directly
- Pilot-operated: solenoid force opens the pilot orifice; main valve is opened by pressure differential
- Power-off reset: spring reset (normally closed) or hold (normally open)

## 5. Hardware Connection

- **Drive current**: DC 24V valves typically draw 200~500 mA; transistor / MOSFET driver required
- **Flyback diode**: reverse-parallel 1N4007 to suppress back-EMF
- **Optocoupler isolation**: recommended in industrial applications
- **Manual reset**: some valves have manual override
- **EMI**: add RC snubber for inductive loads
- **Sealing**: pay attention to IP rating for water inlet valves
- **Temperature**: consider body material for high-temperature environments
- **Explosion-proof**: explosion-proof certification required for gas and similar applications

**Selection suggestions**:

- Beginner / small-flow water valve: direct 12V/24V
- High flow: pilot-operated
- Pneumatic control: SMC 4V210, 4V310 series
- High-precision flow/pressure: proportional valve
- High pressure: hydraulic valve
- Food / medical: food-grade stainless steel valve
- Gas: dedicated gas valve (explosion-proof certification)


## 6. Communication Method

- **GPIO**: direct control (most common)
- **PWM**: proportional valve flow / pressure control
- **Analog voltage**: ±10V or 0~10V
- **Current signal**: 4~20 mA industrial standard
- **Fieldbus**: CANopen, PROFIBUS, DeviceNet
- **IO-Link**: modern industrial bus


## 7. Initialization Flow

1. Configure GPIO / MOSFET driver
2. Set initial state to closed
3. Start control after power is stable
4. Proportional valves need control-parameter configuration (PWM / voltage mapping)


## 8. Driver Interface

```python
class SolenoidValve:
    def __init__(self, pin, active_high=True):
        self.pin = pin
        self.active_high = active_high

    def begin(self):
        self.close()

    def open(self):
        if self.active_high:
            self.pin.high()
        else:
            self.pin.low()

    def close(self):
        if self.active_high:
            self.pin.low()
        else:
            self.pin.high()


class ProportionalValve:
    """Proportional valve."""
    def __init__(self, pwm):
        self.pwm = pwm

    def begin(self):
        pass

    def set_flow(self, percent):
        """0~100."""
        self.pwm.duty(percent)


class LatchingSolenoid:
    """Pulse-type solenoid valve (energy-saving)."""
    def __init__(self, open_pin, close_pin):
        self.open = open_pin
        self.close = close_pin

    def open(self):
        self.open.high()
        time.sleep_ms(50)
        self.open.low()

    def close(self):
        self.close.high()
        time.sleep_ms(50)
        self.close.low()
```

## 10. Error Handling and Exception Recovery

- **Valve stuck**: debris, scaling, rust
- **Leak**: seal aging
- **No response**: coil broken, driver fault
- **Reduced flow**: valve core wear, filter clogging
- **Slow response**: pilot orifice clogged, insufficient pressure differential
- **End of life**: contact oxidation, spring fatigue



- High-pressure gas / liquid is dangerous
- Gas valve leaks may cause explosion; inspect regularly
- Explosion-proof applications must use certified valves
- Medical devices use medical-grade valves



- **No response**: check power and driver circuit
- **Leak**: replace seals
- **Insufficient flow**: check valve core and pressure differential
- **Slow response**: check pilot orifice and filter
- **Bus communication failure**: check address, baud rate, termination resistor


## 11. Low Power

- Holding-type valves: pulse-triggered and held
- Duty-cycle control: intermittent activation saves energy
- Proportional valves: adjust based on demand

## 13. Debugging Methods

1. Listen for the actuation click
2. Measure coil voltage
3. Measure downstream pressure / flow
4. Check sealing (no leak)
5. Long-time temperature check
6. For proportional valves, measure the control voltage with a multimeter


## 14. Frequently Asked Questions

- **No response**: power not connected, driver not working
- **Slow response**: pilot orifice clogged
- **Leak**: seal aging, valve seat wear
- **Insufficient flow**: valve core wear, insufficient pressure differential
- **Overheating**: long-time energization, insufficient heat dissipation
- **Loud noise**: AC drive needs a snubber circuit
