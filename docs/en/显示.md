# Display Overview

Display modules turn digital signals into images readable by the human eye. DriverKit covers **3 display modules**, from small maker-friendly screens to e-ink displays.

## Common Knowledge

- **Driver IC determines interface**: SSD1306 / ST7735 / SSD1351 / RA8875 etc.
- **Interface comparison**:
  - I2C: simplest, slowest (small screens)
  - SPI: high speed, mainstream (medium screens)
  - 8080 / RGB: high speed (large screens)
  - MIPI: ultra-high speed (phone screens)
- **Backlight control**: TFT requires LED constant current, PWM dimming
- **Burn-in prevention**: AMOLED should avoid static images
- **Touch integration**: XPT2046 resistive / GT911 / FT6336 capacitive
