# Brushless Motor (BLDC)


## 1. Module Overview

A Brushless DC motor (BLDC) uses electronic commutation to keep the rotor spinning, with no mechanical wear, long lifetime, high efficiency, and low noise. Common uses:

- Drone propellers
- Power tools (drills, fans, blowers)
- Self-balancing scooters / unicycles
- Electric skateboards
- Hard disk spindles
- Fans and pumps
- Industrial robot joints


## 2. Working Principle and Use Cases

Stator coils are energized to produce a rotating magnetic field that attracts/repels the permanent-magnet rotor. The Electronic Speed Controller (ESC) switches the three-phase winding sequence according to rotor position.

Drive methods:

- **Trapezoidal (6-step)**: switches phase every 60°
- **Sinusoidal**: continuous sinusoidal current, lower torque ripple
- **FOC (Field-Oriented Control)**: vector control, smoothest and most efficient

Rotor position detection:

- **Sensed (with Hall sensor / encoder)**: 3 Hall sensors or encoder
- **Sensorless (Back-EMF)**: estimates rotor position by detecting back-EMF

## 5. Hardware Connection

- Three-phase wires (A/B/C): thick cable (e.g., 14~16 AWG)
- Hall sensor wires: 5 lines (VCC/GND/A/B/C)
- High-current path: thick power cables, low-resistance connectors (XT60, XT90, Amass)
- Heat dissipation: heatsink / fan for long-time high current
- ESD protection: MOSFET gate protection
- ESC mounting: keep away from high-frequency interference (ferrite ring, shielding)
- **Do not power up without load for long periods** — it may damage the motor
- ESC throttle calibration: required every time the motor or ESC changes


## 6. Communication Method

- **PWM**: 50 Hz, 1000~2000 µs (500~2500 µs range)
- **Oneshot125**: single pulse 125~250 µs (8 kHz protocol)
- **Multishot**: 5~25 µs (high speed)
- **DShot150/300/600/1200**: digital protocol (with checksum)
- **ProShot / ProfiLED**: advanced protocols
- **CAN**: high-end ESCs
- **UART**: configuration and telemetry


## 7. Initialization Flow

1. Wiring (power, motor three-phase, signal)
2. ESC throttle calibration (first time or after replacement)
3. Write PWM minimum = lowest speed
4. Set direction (swap any two phases if needed)
5. Configure PID (inside ESC)
6. Start

### 10.1 PWM (standard)

50 Hz, 1 ms = minimum, 2 ms = maximum. Simple but slow response.

### 10.2 Oneshot125

8 kHz protocol, 125~250 µs. Low latency.

### 10.3 DShot

Digital protocol (with checksum):

- DShot150: 150 kbaud
- DShot300: 300 kbaud
- DShot600: 600 kbaud
- DShot1200: 1.2 Mbaud

Requires MCU hardware support (DMA + timer).

### 10.4 BLHeli_32 / AM32

Open-source BLDC ESC firmware, supports DShot, BiDir, etc.



Vector control algorithm; current is decomposed into d/q axes for independent control:

- Torque ripple is minimal
- Highest efficiency (90%+)
- Requires substantial computation (DSP / FPGA / high-speed MCU)
- The open-source SimpleFOC library supports Arduino, STM32, ESP32


## 9. Data Format, Units, and Timing

- PWM: pulse width 1000~2000 µs (typical)
- DShot: 11-bit digital value + 4-bit CRC
- RPM: ESC telemetry (e-RPM)
- Current / voltage: telemetry


## 10. Low Power

- Back-EMF regenerative braking (BLDC regen)
- Active short-circuit braking
- Disable driver when idle

## 12. Debugging Methods

1. Check three-phase wiring (A/B/C mapping)
2. Listen to start-up sound (does it spin normally?)
3. Measure current (no-load 1~2 A, increases under load)
4. Measure temperature (long-time run < 80°C)
5. Adjust PID (inside ESC)
6. Confirm DShot bidirectional telemetry configuration


## 13. Frequently Asked Questions

- **Motor jitters but doesn't turn**: wrong phase order, wrong Hall wiring, ESC not calibrated
- **Hard to start**: sensorless start failure, load too high
- **ESC burnt**: wrong PWM frequency, insufficient cooling, stall over-current
- **Motor overheating**: long-time high current, insufficient cooling
- **Loud noise**: trapezoidal drive, bearing wear
- **High vibration**: poor dynamic balance
- **DShot unresponsive**: wrong wiring, mismatched protocol

## 15. Back-EMF Method (BEMF Sensorless)

Estimate rotor position by detecting back-EMF on the un-energized phase:

- Cannot start from 0 rpm; requires open-loop start
- Stable at medium and high speed
- Complex start algorithm
