# Power Overview

Power chain management is the lifeline of an embedded system. DriverKit covers **3 power modules**, spanning voltage/current monitoring and Li-Ion charging.

## Common Knowledge

- **Battery safety first**: BMS protection board required (over-charge / over-discharge / short circuit)
- **Voltage monitoring**: divider resistor + ADC, pay attention to input impedance
- **Current monitoring**:
  - High-side measurement preserves common ground
  - 4-wire (Kelvin) connection reduces error
- **Four-stage charging**: trickle → constant current → constant voltage → termination
- **PD decoy**: CH224K / HUSB238 control voltage
