# 电子纸（E-Paper / E-Ink）


## 1. 模块概览

电子纸（E-Paper、E-Ink）是一种模拟纸张显示效果的低功耗显示技术。常见于：
- 电子书阅读器（Kindle）；
- 电子价签（ESL）；
- 物流标签；
- 智能手写本；
- 智能手表；
- 公交站牌；
- 户外信息牌；
- 仪表盘。

特点：
- 超低功耗（静态图像 0 功耗）；
- 类纸外观（阳光下可读）；
- 视角 180°；
- 刷新较慢（全刷 1~2 s）；
- 仅黑白（多数），彩色版本较少；
- 可弯曲（柔性屏）。


## 2. 工作原理与适用场景

- **电泳显示技术**：悬浮在液体中的带电粒子（黑/白）在外加电场下移动；
- **微胶囊**：每个像素是一个微胶囊，包含黑/白粒子；
- **电场控制**：黑粒子移到正面 = 显示黑色；白粒子移到正面 = 显示白色；
- 黑色吸光，白色反射环境光；
- **双稳态**：图像保持无需持续供电。

类型：
- **黑白 EPD**（Electrophoretic Display）：最常见；
- **三色 EPD**（黑白红 / 黑白黄）：价签常用；
- **多色 EPD**：ACeP（高级彩色电子纸）4 色 / 7 色；
- **柔性 EPD**：可弯曲。

**选型建议**：
- 价签/标签：2.13"~4.2" 黑白红；

- 低功耗要求：所有 E-Paper（静态 0 功耗）。


## 5. 硬件连接

- 数字电源：3.3V，去耦电容；
- EPD 高电压：±15V、+22V 等（驱动芯片内部升压）；
- SPI：4 线 SCK/MOSI/CS/DC；
- BUSY：EPD 忙碌状态输出（重要）；
- RST：复位；
- 静电防护；
- 不可弯曲（柔性屏除外）；
- 温度限制：低温下刷新失败。
- SPI 速率：典型 10~20 MHz。


## 6. 通信方式

- **SPI**：4 线（SCK/MOSI/MISO/CS），DC 命令/数据选择；
- **I2C**：部分小尺寸屏幕支持；
- 主机：MCU（Arduino、ESP32、STM32、Raspberry Pi）。


## 7. 初始化流程

1. 配置 GPIO（CS/DC/RST/BUSY）；
2. 配置 SPI；
3. 硬件复位（RST 拉低 → 拉高）；
4. 软件复位；
5. 发送驱动初始化序列（厂商提供）；
6. 设置显示窗口（区域）；
7. 发送像素数据；
8. 发送刷新命令；
9. 等待 BUSY 变低（刷新完成）；
10. 进入睡眠（保存功耗）。

### 10.1 全刷

- 时间：1~4 s；
- 效果：无残影；
- 闪烁：会快速闪多次；
- 适合：首次显示、间隔较长的更新。

### 10.2 局部刷

- 时间：0.3~1 s；
- 效果：可能有残影（残影累积）；
- 闪烁：闪烁较少；
- 适合：频繁更新；
- 注意：每 N 次局部刷后做一次全刷去残影。

### 10.3 快刷

- 部分屏幕支持超快刷（0.1~0.3 s）；
- 残影严重，仅适合短时显示。



- ASCII 字符：5x8 / 8x16 / 12x16；
- 中文字库：需 Flash 存字模（GBK、Unicode）；
- 位图：BMP 单色转换；
- LVGL 支持矢量字体，但慢刷限制实用性。



- 局部刷新多次后累积残影；
- 定期全刷去残影（如每 10 次局部刷 1 次全刷）；
- 部分屏幕支持自动去残影。


## 9. 数据格式与单位

- 1 bit/pixel（黑白）；
- 黑白红 EPD：1 bit 黑白 + 1 bit 红（双缓冲）；
- ACeP：1 byte/pixel（8 色）；
- 字节序：MSB 在前（按行扫描）。


## 10. 中断与 DMA

- BUSY 信号：刷新时为高，完成后为低；
- 中断：BUSY 下降沿触发。


## 11. 错误处理

- **BUSY 一直高**：SPI 时序错、刷新命令错；
- **黑屏**：电源、复位错；
- **残影严重**：未定期全刷；
- **刷新失败**：温度低（< 0°C）；
- **数据错**：SPI 速率过快。


## 12. 低功耗

电子纸核心优势：

- 静态显示：0 功耗（双稳态）；
- 仅刷新时耗电（20~40 mA·s）；
- 刷新后进入睡眠（< 10 µA）；
- 电池供电可工作数年（每天几次刷新）。

```python
epd.sleep()  # 刷新后进入睡眠
# 长时间保持显示，0 功耗
```


## 13. 示例代码

```python
epd = EPD(spi, dc_pin, rst_pin, busy_pin, width=296, height=128)
epd.begin()
epd.clear()
epd.draw_text(0, 0, "Hello E-Paper!")
epd.show(mode=0)  # 全刷
time.sleep_ms(5000)
epd.sleep()  # 节省功耗
```


## 14. 调试方法

1. 检查电源电压（含高压）；
2. 测 BUSY 信号时序；
3. 用示波器看 SPI 波形；
4. 跑厂商示例代码；
5. 测试不同温度下刷新；
6. 检查残影情况。


## 15. 常见问题

- **黑屏**：电源、复位、初始化失败；
- **残影严重**：未定期全刷；
- **局部刷失败**：部分屏幕不支持；
- **低温柔刷新失败**：温度超出范围；
- **部分显示**：窗口设置错；
- **功耗高**：未调用 sleep；
- **模糊**：温度、湿度影响。


## 16. 参考资料
