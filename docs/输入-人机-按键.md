# 按键


## 1. 模块概览

按键（Button/Switch）是最基本的人机输入元件。常见于：
- 家电控制面板（电视、空调、洗衣机）；
- 工业控制面板；
- 仪器仪表；
- 遥控器；
- 玩具、游戏手柄；
- 键盘、鼠标；
- 嵌入式设备（Reset 按键）。

类型：
- **机械按键**（轻触开关）：瞬时接触，按下导通，松开断开；
- **自锁开关**：按下后保持，再次按下释放；
- **滑动开关**：拨动开关，状态保持；
- **薄膜开关**：PCB 上的铜箔按键；
- **电容触摸按键**：见电容触摸文档；
- **旋转编码器**：见旋转编码器文档。


## 2. 工作原理与适用场景

机械按键通过金属弹片接触导通电流：
- 导通（按下）：触点闭合；
- 断开（松开）：触点弹开；
- 抖动：金属弹片接触瞬间产生多次开闭（几 ms ~ 几十 ms）；
- 寿命：10⁵~10⁷ 次（金属弹片疲劳）。

## 5. 硬件连接

### 5.1 基本电路

- **上拉电阻**：按键一端接 VDD（通过电阻），另一端接 MCU；
- **下拉电阻**：按键一端接 GND（通过电阻），另一端接 MCU；
- 内部上拉：多数 MCU 有内部上拉（30~50 kΩ），可省外部电阻；
- 按下：GPIO 低电平（按下低）或高电平（按下高）；
- 串联电阻：100Ω 限流保护 MCU。

### 5.2 矩阵键盘

- 行线：MCU 输出（拉低扫描）；
- 列线：MCU 输入（上拉）；
- 二极管：防串键（同时按多个按键）；
- 8 根线可扫描 4x4 = 16 键。

### 5.3 防抖

- 硬件 RC：10 nF 电容 + 1 kΩ 电阻 = 10 µs 时间常数；
- 硬件去抖 IC：MAX6816 等；
- 软件去抖：见下文。

### 5.4 静电保护

- TVS 二极管；
- 100Ω 串联电阻；
- 滤波电容。


## 6. 通信方式

- 单 GPIO：单按键；
- 矩阵：行列扫描；
- I2C/SPI：触摸 IC；
- USB HID：键盘（详见 USB 文档）；
- 蓝牙 HID：无线键盘；
- 红外：遥控器（NEC 协议等）。


## 7. 初始化流程

1. 配置 GPIO（输入 + 上下拉）；
2. 配置中断（上升/下降沿）；
3. 配置定时器（去抖、扫描）；
4. 启动。

class Button:
    """单个按键。"""
    def __init__(self, pin, active_low=True, debounce_ms=20):
        self.pin = pin
        self.active_low = active_low
        self.debounce_ms = debounce_ms
        self.pressed = False
        self.last_state = False
        self.last_time = 0

    def begin(self):
        pass

    def is_pressed(self):
        raw = self.pin.value()
        current = (not raw) if self.active_low else raw
        return current and self._stable(current)

    def _stable(self, state):
        now = time.ticks_ms()
        if state != self.last_state:
            self.last_state = state
            self.last_time = now
            return False
        return time.ticks_diff(now, self.last_time) > self.debounce_ms

- **单击**：按下后松开（< 200ms 内）；
- **双击**：短时间内连续两次单击；
- **长按**：按下持续超过 N 秒（1~3 秒）；
- **超长按**：持续超过 N 秒（如 5 秒）；
- **按住**：持续按住中。

实现方法：状态机。



```python
class MultiButton:
    def __init__(self, pin, long_press_ms=1000):
        self.pin = pin
        self.long_press_ms = long_press_ms
        self.press_start = None

    def update(self):
        current = self.pin.value()
        if current and self.press_start is None:
            # 按下
            self.press_start = time.ticks_ms()
        elif not current and self.press_start is not None:
            # 松开
            duration = time.ticks_diff(time.ticks_ms(), self.press_start)
            self.press_start = None
            if duration > self.long_press_ms:
                return "LONG"
            return "CLICK"
        return None
```



```python
class MatrixKeypad:
    def __init__(self, row_pins, col_pins):
        self.row_pins = row_pins
        self.col_pins = col_pins

    def scan(self):
        keys = []
        for r in range(len(self.row_pins)):
            self.row_pins[r].low()  # 拉低当前行
            for c in range(len(self.col_pins)):
                if self.col_pins[c].value() == 0:
                    keys.append((r, c))
            self.row_pins[r].high()
        return keys
```

## 10. 校准与滤波

```python
def debounce(pin, last_state, last_time, ms=20):
    current = pin.value()
    if current != last_state:
        if time.ticks_diff(time.ticks_ms(), last_time) > ms:
            return current, current, time.ticks_ms()
        return None, last_state, last_time
    return None, last_state, last_time
```


## 11. 中断与 DMA

- 单按键：中断（按下/松开）+ 定时器去抖；
- 矩阵键盘：定时器周期扫描（如 10 ms）；
- 长按：定时器跟踪按下时长。


## 12. 错误处理

- **抖动**：硬件/软件去抖；
- **串键（矩阵）**：多个键同时按下，二极管防串或软件消解；
- **意外触发**：静电干扰，加去耦和限流；
- **粘连**：机械按键老化，需要更换；
- **触点氧化**：低电流负载失效，使用金触点或"湿电路"。


## 13. 低功耗

- 按键中断唤醒 MCU（从深度休眠）；
- 内部上拉电阻很省电；
- 不使用时关闭扫描定时器。

## 15. 调试方法

1. 用示波器看按键波形（机械抖动）；
2. 用逻辑分析仪记录按下/松开时间；
3. 测 GPIO 电平变化；
4. 测接触电阻；
5. 测试寿命（按键寿命测试仪）。


## 16. 常见问题

- **误触发**：抖动未消除，静电干扰；
- **不响应**：GPIO 配置错，上下拉未配置；
- **双击无效**：去抖时间设置过长；
- **长按太敏感**：时长阈值设置不当；
- **矩阵串键**：缺二极管；
- **机械卡死**：按键老化或损坏。
