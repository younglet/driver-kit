# 直流电机


## 1. 模块概览

直流电机（DC Motor）通过直流电压驱动旋转，是最常见的旋转执行器。广泛用于：
- 智能小车驱动轮；
- 玩具；
- 电动工具（电钻、风扇）；
- 家电（洗衣机、吸尘器）；
- 风扇、泵；
- 机器人关节（需配编码器闭环）。

类型：
- **有刷直流电机（BDC）**：经典、成本低、有刷磨损；
- **无刷直流电机（BLDC）**：电子换向、长寿命、效率高（详见无刷电机文档）；
- **空心杯电机**：高速无铁芯、低惯量；
- **杯式电机**：微小型；
- **伺服电机**：含编码器闭环（详见伺服电机文档）。


## 2. 工作原理与适用场景

通电导线在磁场中受力（左手定则）。电刷+换向器让电枢自动切换方向，使转子持续旋转。

转速公式：

```text
n = (U - I × Ra) / k_e
U = 电源电压
I = 电枢电流
Ra = 电枢电阻
k_e = 反电动势常数
```

调速：
- **PWM 调速**：改变平均电压（占空比）；
- **H 桥换向**：改变旋转方向；
- **串联电阻**：简单但低效；
- **降压调速**：线性稳压，效率低。

## 5. 硬件连接

- H 桥驱动：必须使用 H 桥驱动芯片（如 TB6612、L298N、BTS7960、DRV8871）；
- **H 桥禁止直连！** 直连会导致电源短路；
- 续流二极管：内置于驱动芯片，但外部电源需加 TVS/肖特基；
- 去耦电容：100nF + 100µF；
- 散热：大电流电机需要散热片；
- 反向 EMF：电机突然停止时产生高压尖峰，需续流回路；
- PWM 频率：1 kHz ~ 25 kHz 通用；
- 死区时间：避免 H 桥上下桥臂直通；
- 编码器：闭环控制需要 AB 相编码器。


## 6. 通信方式

- **PWM 信号**：调节转速（占空比 0~100%）；
- **H 桥方向控制**：两路 GPIO（IN1/IN2 或 PWM + DIR）；
- **模拟电压**：少数驱动板（如 BTS7960）支持模拟电压调速；
- **编码器反馈**：AB 相脉冲输入。


## 7. 初始化流程

1. 配置 GPIO（IN1/IN2/PWM 输出）；
2. 配置 PWM 模块（频率 20 kHz）；
3. 初始化为停止状态（IN1=IN2=0）；
4. 配置编码器输入（如使用）；
5. 设置 PID 参数（闭环）；
6. 启动。


## 8. 驱动接口

```python
class DCMotor:
    def __init__(self, in1, in2, pwm):
        self.in1 = in1
        self.in2 = in2
        self.pwm = pwm
        self.encoder = None

    def begin(self):
        pass

    def set_speed(self, percent):
        """-100~+100，正反转和速度。"""
        if percent > 0:
            self.in1.high(); self.in2.low()
            self.pwm.duty(percent)
        elif percent < 0:
            self.in1.low(); self.in2.high()
            self.pwm.duty(-percent)
        else:
            self.in1.low(); self.in2.low()
            self.pwm.duty(0)

    def stop(self):
        self.in1.low(); self.in2.low()
        self.pwm.duty(0)

    def brake(self):
        """短路制动。"""
        self.in1.high(); self.in2.high()
        self.pwm.duty(100)

    def set_encoder(self, encoder):
        self.encoder = encoder

    def velocity(self):
        """返回 rpm。"""
        if self.encoder:
            return self.encoder.read_velocity()
        return 0
```



```python
def pid_control(target_rpm, current_rpm, kp, ki, kd, integral, prev_err):
    err = target_rpm - current_rpm
    integral += err
    derivative = err - prev_err
    output = kp * err + ki * integral + kd * derivative
    return output, integral, err
```

典型 PID 参数：kp=0.5, ki=0.1, kd=0.05（需现场调节）。


## 9. 数据格式与单位

- PWM 占空比：0~100%（或 0~65535）；
- 转速：rpm 或 deg/s；
- 电流：A；
- 扭矩：N·m；
- 编码器计数：脉冲数。


## 10. 校准与滤波

- 编码器数据：中值滤波去尖峰；
- PWM 输出：硬件滤波（电机线圈本身）；
- 电流采样：RC 低通滤波。


## 11. 中断与 DMA

- 编码器中断：捕获 A/B 相信号；
- 定时器中断：PID 周期（如 1 ms）；
- DMA：编码器计数（减少 CPU 负担）。


## 12. 错误处理

- **堵转**：电流急剧增大，触发过流保护；
- **过压**：反向 EMF 尖峰，加续流回路；
- **过热**：超过额定电流，散热或降功率；
- **失步**：闭环系统 PID 不当；
- **编码器失效**：备份值或软件重启。


## 13. 低功耗

- PWM 调速比电阻降压效率高；
- 空闲时停止电机（0% PWM）；
- 短时降速节能。

    def __init__(self, in1, in2, pwm):
        self.in1 = in1
        self.in2 = in2
        self.pwm = pwm

    def begin(self):
        self.stop()

    def forward(self, speed):
        """speed 0~100。"""
        self.in1.high()
        self.in2.low()
        self.pwm.duty(speed)

    def backward(self, speed):
        self.in1.low()
        self.in2.high()
        self.pwm.duty(speed)

    def stop(self):
        self.in1.low()
        self.in2.low()
        self.pwm.duty(0)

## 15. 调试方法

1. 缓慢增加 PWM 占空比观察电机启动；
2. 测试正反转切换；
3. 测量空载电流和堵转电流；
4. PID 调试：先 P，逐步加 I/D；
5. 长时间运行观察发热情况。


## 16. 常见问题

- **电机不转**：电源未接、H 桥逻辑错、PWM 未输出；
- **PWM 但不转**：H 桥死区时间太长，电流不够；
- **抖动**：PID 参数不当或编码器信号噪声；
- **过热**：长时间堵转或超额定电流；
- **反转但不能调速**：PWM 频率太低（如 100 Hz 内电机啸叫）。
