# 气压传感器


## 1. 模块概览

气压传感器测量环境大气压力，常用于高度计、气象站、无人机定高、GPS 辅助、室内导航、登山/户外设备、手机气压计。

海平面标准气压约 1013.25 hPa，每升高 8 米气压下降约 1 hPa（高度与气压非线性）。


## 2. 工作原理与适用场景

- **MEMS 压阻式**：硅膜片受压变形导致电阻变化；
- **MEMS 电容式**：膜片间距变化导致电容变化（低功耗高精度）；
- **压电式**：声波共振频率变化（高精度工业级）。

应用：
- 无人机/弹道定高；
- 户外手表高度计；
- 室内楼层定位（与地图匹配）；
- 气象站；
- 真空/密封设备检漏；
- 医疗呼吸机。

## 5. 硬件连接

- I2C 上拉：4.7kΩ；
- BMP280/BME280 默认地址 0x76（SDO=GND）或 0x77（SDO=VDD）；
- 避免振动干扰：MEMS 对振动敏感；
- 模块开孔：让气压能感知环境；
- 防水版本有 PTFE 膜片，注意孔位布局；
- 远离 DC-DC、电机等压力源。


## 6. 通信方式

- I2C：标准 / 快速模式；
- SPI：4 线，模式 0 或 3；
- 数据包：压力 + 温度原始值 + 校准系数。


## 7. 初始化流程

1. 读取 WHO_AM_I（0x58 / 0x60）；
2. 软复位；
3. 读取校准系数（出厂保存在 NVM）；
4. 设置工作模式（sleep / forced / continuous）；
5. 设置压力和温度采样精度（OSR）；
6. 设置输出数据率；
7. 设置滤波器；
8. 触发测量（如 forced 模式）；
9. 读取原始数据并补偿计算。

### 10.1 国际标准大气（ISA）模型

```text
P = P0 * (1 - L*h/T0)^(g*M/(R*L))
```

简化公式：

```python
import math

def altitude_from_pressure(p, p0=1013.25):
    """简化公式，精度 ±10m。"""
    return 44330.0 * (1 - (p / p0) ** (1 / 5.255))
```

### 10.2 精度提升

使用当地实时气压（从气象站或 GPS）作为 P0 比标准值更准。

### 10.3 温度补偿

高精度需求考虑温度对密度的影响，可使用完整 Barometric Formula。


## 9. 数据格式与单位

- 单位：Pa（帕斯卡）或 hPa（百帕）；
- 1 hPa = 100 Pa；
- 标准海平面气压：1013.25 hPa；
- 温度：°C；
- 数据格式：原始值 + 校准系数 → 补偿计算 → 实际值。


## 10. 校准与滤波

- 工厂校准系数已存 NVM，无需用户校准；
- 高精度需求：使用已知气压基准（如气象站数据）校准；
- 海平面参考：用本地气象站数据；
- 长期漂移：通常每年 < 1 hPa，可忽略。


## 11. 中断与 DMA

- 部分芯片支持数据就绪中断；
- FIFO：少量芯片支持（MS5611 无 FIFO，但读得快）；
- 单次测量推荐 forced 模式。


## 12. 错误处理

- WHO_AM_I 错：I2C 地址错、上拉未接；
- 数据溢出：超出量程（如低气压环境 < 300 hPa）；
- 读数跳变：振动干扰、孔位阻塞；
- 海拔漂移：温度变化、湿度变化导致。


## 13. 低功耗

- sleep 模式 < 1µA；
- forced 模式按需触发；
- 间歇测量：1 Hz 已足够户外高度计。

## 15. 调试方法

1. 读取 WHO_AM_I 验证；
2. 室内读数与气象站数据对比；
3. 上下楼梯观察高度变化；
4. 加热/冷却传感器观察温度响应；
5. 长时间监测零偏稳定性。


## 16. 常见问题

- **高度漂移**：温度变化影响气压，使用温度补偿或本地 P0；
- **数据跳变**：孔位堵塞（防尘网堵）；
- **BMP280/BME280 错**：注意 BME280 是 0x60，BMP280 是 0x58；
- **海拔误差大**：未使用本地气压作为参考；
- **凝结水损坏**：湿度大时膜片凝水，需防水型号。
