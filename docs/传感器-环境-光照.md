# 光照传感器


## 1. 模块概览

光照传感器测量环境光的强度（lux）或可见光 + 红外光分量，常用于：
- 自动亮度调节（手机、平板、笔记本）；
- 路灯/楼道灯自动开关；
- 摄像头白平衡/曝光辅助；
- 温室光照监测；
- 显示背光调节；
- 太阳能跟踪。

类型：
- **光敏电阻（LDR）**：CdS 元件，电阻随光强变化；
- **光电二极管**：输出电流与光强成正比；
- **光电三极管**：放大后的电流输出；
- **集成 IC**：BH1750、TSL2561、TSL2591、MAX44009、VEML7700 等，内置 ADC + 红外滤除。


## 2. 工作原理与适用场景

- **LDR**：CdS 材料暗电阻 ~1MΩ，亮电阻 ~1kΩ。需 ADC 采样。
- **光电二极管**：反向偏置时电流随光强变化，跨阻放大输出。
- **集成 IC**：硅光电二极管阵列 + ADC + I2C 输出 lux 值，自带红外滤除，准确性高。

应用：
- 自动调光（手机背光、车载显示屏）；
- 路灯/楼道灯自动控制；
- 智能窗帘/卷帘；
- 智能家居（环境感知）；
- 农业温室光合有效辐射（PAR）监测；
- 工业流水线产品识别。

## 4. 关键参数与选型

| 参数 | 说明 |
|---|---|
| 量程 | 0~100k lux（户外阳光）~ 0~10k lux（室内） |
| 分辨率 | 16/20/24 bit |
| 精度 | ±10%~±20%（多数） |
| 滤除红外 | 影响室内灯与人造光准确度 |
| 输出数据率 | 1 Hz~几百 Hz |
| 接口 | I2C（多数）、模拟 |
| 电源 | 1.8V/3.3V |
| 功耗 | 测量 µA 级，待机 nA 级 |
| 视场角 | 50°~120° |
| 光谱响应 | 接近人眼 555 nm 灵敏度（V-lambda 滤波） |

**选型建议**：
- 入门/简单：光敏电阻 + ADC；
- 室内自动调光：BH1750、VEML7700、OPT3001；
- 高动态：TSL2591、VEML6035、OPT4048；
- 颜色识别：TCS34725、APDS-9960；
- 光谱：AS7341；
- 超低功耗：MAX44009、OPT3002。


## 5. 硬件连接

- I2C 上拉：4.7kΩ；
- ADDR 引脚：多数芯片允许修改 I2C 地址 LSB；
- INT：阈值中断输出；
- 透光窗口：避免被遮挡；
- 暗电流：光敏电阻需要 100kΩ 量级的分压电阻；
- 滤光片：VEML7700 等内置 IR 滤除；
- 注意：方向性强（侧装或贴装差异较大）。


## 6. 通信方式

- 模拟输出：电压信号，ADC 采样；
- I2C：标准 100/400 kHz；
- 部分芯片支持 SMBus Alert 中断；
- 集成 IC 输出 lux 或 raw 计数。


## 7. 初始化流程

1. 配置 I2C；
2. 读 WHO_AM_I；
3. 软复位；
4. 设置测量模式（单次 / 连续）；
5. 设置积分时间 / 增益；
6. 启动测量；
7. 读取 lux 或 raw 值。

## 9. 数据格式与单位

- 照度单位：lux（勒克斯）；
- 1 lux = 1 lm/m²；
- 室外阳光约 10万~10万 lux；
- 室内照明约 100~500 lux；
- 月光约 0.1~1 lux；
- 不同光源：白炽灯 555 nm 灵敏度与人眼接近，LED 蓝光偏多。


## 10. 校准与滤波

- 集成 IC 工厂校准；
- 高精度需求：使用已知照度源（光强计）校准；
- 注意：手机屏幕自动亮度使用本地算法，可能与传感器差异；
- 温度影响：少量。


## 11. 中断与 DMA

- 阈值中断：低/高阈值越界触发 INT；
- 部分芯片支持持久中断（需要持续 N 周期越界）；
- 自动调光：设不同档位阈值。


## 12. 错误处理

- I2C 错误：检查地址、上拉；
- 数据饱和：户外强光；
- 读数 0：环境全黑或传感器故障；
- 漂移：温度、老化。


## 13. 低功耗

- 单次测量后休眠；
- 待机 < 1µA；
- 1 秒 1 次测量足够大多数场景。

## 15. 调试方法

1. 室内遮光与开灯对比读数；
2. 手遮光观察 lux 下降；
3. 阳光下读数（注意量程）；
4. 对照手机照度计 App 验证；
5. 用手电筒照射观察瞬间响应。


## 16. 常见问题

- **BH1750 读数不稳**：积分时间未配置；
- **户外光饱和**：量程不够，加衰减滤片；
- **TSL2561 读数偏大**：红外未滤除，使用双通道法；
- **I2C 错误**：ADDR 引脚电平错；
- **光敏电阻响应慢**：物理特性，几百毫秒。

## 18. 算法

### 11.1 V-lambda 校正

人眼对 555 nm 绿光最敏感，传感器输出应近似 V(λ) 函数。

### 11.2 红外补偿

典型光源含红外成分，室内自动调光需要 IR 滤除或补偿算法（TSL2561 双通道法）。

### 11.3 双通道法

VIS + IR 双通道 → 估算可见光 → 滤除红外干扰。
