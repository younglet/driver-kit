# 输入总览

人机交互输入设备。DriverKit 收录 **3 个输入模块**，覆盖最基础的人机交互元件。

## 模块速查

| 模块 | 说明 | 文档 |
|---|---|---|
| 按键 | 机械 / 触摸 / 矩阵键盘 | [打开](./输入-人机-按键) |
| 旋转编码器 | 增量 / 绝对值旋钮与角度 | [打开](./输入-人机-旋转编码器) |
| 摇杆 | 双轴电位器 / 霍尔摇杆 | [打开](./输入-人机-摇杆) |

| 应用场景 | 推荐 |
|---|---|

## 共性知识

- **去抖必须**：硬件 RC 或软件延时（5~30 ms）
- **长按 / 双击**：状态机实现多功能识别
- **矩阵扫描**：扫描周期 10 ms 左右
- **上拉 / 下拉**：浮空输入易受干扰
- **工业抗干扰**：TVS、屏蔽线、电平转换
