# 显示总览

显示模块把数字信号变成人眼能看懂的图像。DriverKit 收录 **3 个显示模块**，覆盖小尺寸创客屏到电子墨水屏。

## 模块速查

| 模块 | 说明 | 文档 |
|---|---|---|
| OLED | 自发光、高对比、薄（0.49"~6.5"） | [打开](./显示-图形-OLED) |
| TFT 液晶 | 全彩高速（0.96"~10.1"，IPS 广视角） | [打开](./显示-图形-TFT液晶) |
| 电子纸 | 静态 0 功耗，阳光下可读（1"~13.3"） | [打开](./显示-图形-电子纸) |

## 选型速查表

| 应用场景 | 推荐 |
|---|---|
| 智能手表 | [OLED](./显示-图形-OLED) AMOLED（高 PPI） |
| 创客 / Arduino / ESP32 | [OLED](./显示-图形-OLED) 0.96" / 1.3"（I2C/SPI） |
| 智能家居面板 | [TFT 液晶](./显示-图形-TFT液晶) 2.4"/3.5"（带触摸） |
| 工业 HMI | [TFT 液晶](./显示-图形-TFT液晶) 4.3"/5"/7"（IPS + 触摸） |
| 电子价签 | [电子纸](./显示-图形-电子纸) 2.13"/2.9"/4.2"（黑白红） |
| 电子书 | [电子纸](./显示-图形-电子纸) 7.5"/10.3"/13.3" |
| 户外信息牌 | [电子纸](./显示-图形-电子纸)（阳光下可读） |
| 智能手写本 | [电子纸](./显示-图形-电子纸) 10.3"+ |

## 共性知识

- **驱动 IC 决定接口**：SSD1306 / ST7735 / SSD1351 / RA8875 等
- **接口对比**：
  - I2C：最简单，最慢（小屏）
  - SPI：高速，主流（中屏）
  - 8080 / RGB：高速（大屏）
  - MIPI：超高速（手机屏）
- **背光控制**：TFT 必须 LED 恒流，PWM 调光
- **防烧屏**：AMOLED 避免静止画面
- **触摸集成**：XPT2046 电阻 / GT911 / FT6336 电容
