# 红外避障传感器


## 1. 模块概览

红外避障传感器使用红外对管（一个红外 LED + 一个光电二极管或三极管），用于：
- 智能小车避障；
- 机器人循迹；
- 工业产品计数；
- 黑线循迹；
- 物体接近检测；
- 自动门感应。

特点：成本极低（¥1~5）、响应快，但仅适合近距（几厘米内）、定性检测。


## 2. 工作原理与适用场景

红外 LED 发射红外光，光电三极管接收反射光：
- 无反射物 → 三极管截止 → 输出高电平；
- 有反射物 → 反射光进入三极管 → 导通 → 输出低电平（取决于电路）。

检测距离与反射物表面颜色、材质、距离相关：
- 白色物体：反射强，距离较远；
- 黑色物体：反射弱，距离较近；
- 表面纹理、倾斜角影响。

**选型建议**：
- 小车避障：通用模块 / FC-03 / TCRT5000；
- 循迹：TCRT5000 / QTR 系列；
- 物体计数：E18-D80NK / 槽型对管；
- 工业检测：M18 圆柱光电开关；
- 自适应阈值：QTR-3A、QTR-8A（模拟输出）。


## 5. 硬件连接

- 模拟输出：直接 ADC 采样；
- 数字输出：通过比较器（LM393）调节阈值；
- TCRT5000 模块：VCC、GND、DO（数字）、AO（模拟）；
- 注意：环境光（阳光、白炽灯）会影响读数，使用调制红外光（38 kHz）可抗干扰；
- 红外 LED 限流电阻：通常 220Ω~1kΩ；
- 检测距离：白色物体 5 cm，黑色物体 1 cm。


## 6. 通信方式

- 模拟输出：电压信号（0~VCC）；
- 数字输出：高/低电平（比较器阈值可调）；
- 无标准协议。


## 7. 初始化流程

1. 配置 GPIO（输入模式）；
2. 配置 ADC（如读取模拟输出）；
3. 启动周期性采样；
4. 数字输出直接判定；
5. 模拟输出根据阈值判定（程序或自适应）。


## 8. 驱动接口

```python
class IRProximity:
    def __init__(self, pin, adc=None):
        self.pin = pin  # 数字
        self.adc = adc  # 模拟（可选）

    def begin(self):
        pass

    def is_obstacle(self):
        """返回是否有障碍物（数字输出）。"""
        return self.pin.value() == 0  # 假设低电平有效

    def read_raw(self):
        """返回 ADC 原始值。"""
        return self.adc.read() if self.adc else None

    def read_reflectance(self):
        """0=白 1=黑，根据阈值。"""
        return self.read_raw() < self.threshold
```

## 9. 数据格式与单位

- 数字输出：布尔（高/低电平）；
- 模拟输出：电压或 ADC 计数；
- 单位：无标准距离单位（仅定性）。


## 10. 校准与滤波

- 数字输出：通过模块上的电位器调节；
- 模拟输出：通过实验确定阈值（白/黑物体的 ADC 值）。

### 10.1 自适应阈值

```python
def auto_calibrate(samples):
    """根据场景自动校准阈值。"""
    max_val = max(samples)
    min_val = min(samples)
    return (max_val + min_val) / 2
```



- 数字输出：可多次采样去抖；
- 模拟输出：中值滤波去尖峰；
- 滑动平均降低抖动。


## 11. 中断与 DMA

- 数字输出可用于中断唤醒；
- 周期性采样 10~100 Hz 足够。


## 12. 错误处理

- 检测距离超出范围：始终返回固定值；
- 阳光直射：可能误触发（调制光可缓解）；
- 黑色物体：检测距离极短，需更换传感器（激光）；
- 模块失效：LED 烧毁。


## 13. 低功耗

- 关闭 LED 可大幅降低功耗；
- 检测时打开（间歇工作）。


## 14. 示例代码

```python
class IRProximity:
    def __init__(self, do_pin, ao_pin=None):
        self.do = do_pin
        self.ao = ao_pin
        self.threshold = 512

    def begin(self):
        pass

    def is_obstacle(self):
        """低电平有效。"""
        return self.do.value() == 0

    def read_ao(self):
        """返回 0~1023。"""
        return self.ao.read()

    def read_color(self):
        """黑白判定。"""
        if self.ao.read() < self.threshold:
            return "black"
        return "white"


ir = IRProximity(do_pin, ao_pin)
while True:
    print("障碍物:", ir.is_obstacle(), "颜色:", ir.read_color())
```


## 15. 调试方法

1. 用白纸接近测试数字输出翻转距离；
2. 用黑纸测试数字输出翻转距离；
3. 检查模拟输出变化范围；
4. 阳光下测试（看是否误触发）；
5. 调节电位器看阈值变化。


## 16. 常见问题

- **数字输出频繁跳变**：阈值不稳，使用多次采样去抖；
- **黑色物体检测不到**：检测距离短，换激光 ToF；
- **阳光下误触发**：使用 38 kHz 调制或加滤光片；
- **多个传感器相互干扰**：错开发射时序；
- **TCRT5000 检测距离过近**：更换大功率 LED 或使用其他型号。
