# 通信总览

通信是连接 MCU 与外设、设备与设备、终端与云端的桥梁。DriverKit 收录 **12 个通信模块**，按形态划分为 **总线协议** 和 **协议 / 无线** 两类。

## 按领域速查

### 总线（7 个）

芯片间 / 板内的有线总线。

| 模块 | 说明 | 文档 |
|---|---|---|
| GPIO | 最基础的输入输出与扩展 | [打开](./通信-总线-GPIO) |
| I2C | 两线制同步总线，传感器主流 | [打开](./通信-总线-I2C) |
| SPI | 高速全双工，Flash / 显示首选 | [打开](./通信-总线-SPI) |
| UART | 异步串口，调试与模块通信 | [打开](./通信-总线-UART) |
| RS485 | 工业总线，Modbus 物理层 | [打开](./通信-总线-RS485) |
| CAN | 汽车与工业实时总线 | [打开](./通信-总线-CAN) |
| USB | 通用串行总线，PC 与充电 | [打开](./通信-总线-USB) |

### 协议 / 无线（5 个）

应用层协议和无线通信。

| 模块 | 说明 | 文档 |
|---|---|---|
| Modbus | 工业事实标准协议（RTU/TCP） | [打开](./通信-协议-Modbus) |
| 蓝牙 BLE | 低功耗短距离（可穿戴/IoT） | [打开](./通信-无线-蓝牙BLE) |
| Wi-Fi | 高速局域网（IoT/视频） | [打开](./通信-无线-WiFi) |
| LoRa | 远距离低功耗（LPWAN） | [打开](./通信-无线-LoRa) |
| GNSS | GPS / 北斗等卫星定位 | [打开](./通信-无线-GNSS) |

## 选型速查表

| 需求 | 推荐 |
|---|---|
| 接传感器 | [I2C](./通信-总线-I2C)（多设备）或 [SPI](./通信-总线-SPI)（高速） |
| 调试串口 | [UART](./通信-总线-UART) + USB-TTL 桥接 |
| 多设备板内 | [I2C](./通信-总线-I2C)（注意地址冲突） |
| 工业现场 | [RS485](./通信-总线-RS485) + [Modbus](./通信-协议-Modbus) |
| 汽车电子 | [CAN](./通信-总线-CAN)（首选） |
| 视频传输 | [Wi-Fi](./通信-无线-WiFi) 或 [USB](./通信-总线-USB) |
| 可穿戴设备 | [蓝牙 BLE](./通信-无线-蓝牙BLE) |
| 远程监测 | [LoRa](./通信-无线-LoRa) 或 [NB-IoT](../) |
| 定位 / 授时 | [GNSS](./通信-无线-GNSS) |
| PC 通信 / 充电 | [USB](./通信-总线-USB) |

## 共性知识

- **电平匹配**：3.3V MCU 接 5V 设备需电平转换
- **终端电阻**：RS485/CAN 总线两端 120Ω
- **CRC / 校验**：所有协议都有错误检测
- **总线仲裁**：CAN 按 ID 仲裁，I2C/UART 单主机
- **天线**：RF 模块（BLE/Wi-Fi/LoRa/GNSS）必须正确匹配
