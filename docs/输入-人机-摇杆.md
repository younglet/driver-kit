# 摇杆


## 1. 模块概览

摇杆（Joystick）是用于多自由度输入的设备。常见于：
- 游戏手柄；
- 遥控设备（无人机、机器人）；
- 工业遥控（起重机、AGV）；
- 医疗器械（CT、MRI 操作）；
- 模拟驾驶；
- 平衡车/独轮车控制；
- 体感控制。

类型：
- **双轴电位器摇杆（PS2 Joystick）**：经典创客；
- **三轴（带 Z 按键）**：X/Y 模拟 + Z 数字按键；
- **霍尔摇杆**：磁感应，无机械磨损；
- **电容摇杆**：高分辨率；
- **光电摇杆**：高精度；
- **力反馈摇杆**：带力矩反馈（高端）；
- **力传感器摇杆**：压力感应。


## 2. 工作原理与适用场景

### 2.1 电位器摇杆

- X 轴和 Y 轴各一个电位器；
- 中心位置输出中点电压（如 2.5V@5V）；
- 移动改变电阻比例；
- 输出电压经 ADC 读取。

### 2.2 霍尔摇杆

- 操纵杆下方有磁铁；
- 磁敏感元件（霍尔/AMR/TMR）检测磁场方向；
- 输出与位置成比例；
- 无机械磨损，寿命长。

### 2.3 Z 轴按键

- 摇杆按下时触发开关；
- 通常 SW 引脚低电平表示按下。

## 5. 硬件连接

### 5.1 基本电路

- VRx：X 轴模拟输出 → MCU ADC；
- VRy：Y 轴模拟输出 → MCU ADC；
- SW：按键 → GPIO（输入上拉）；
- VCC：3.3V/5V；
- GND：地。

### 5.2 电位器规格

- 多数为 10 kΩ；
- 中心位置电压 = VCC / 2；
- 最小/最大位置电压 = GND / VCC；
- 测量范围：0~VCC。

### 5.3 滤波

- 电容（100nF）并联到 VCC；
- 软件中值滤波（去除尖峰）；
- 软件滑动平均（降噪）；
- 死区：中心范围直接输出 0。


## 6. 通信方式

- 模拟电压：VRx、VRy（最常见）；
- I2C：霍尔 + 专用 IC；
- SPI：霍尔 + 专用 IC；
- PWM：少数；
- USB HID：游戏手柄。


## 7. 初始化流程

1. 配置 ADC（VRx、VRy）；
2. 配置 GPIO（SW 输入上拉）；
3. 读取中心位置（calibration）；
4. 设置死区阈值；
5. 启动读取。


## 8. 驱动接口

```python
class Joystick:
    def __init__(self, vrx_adc, vry_adc, sw_pin):
        self.vrx = vrx_adc
        self.vry = vry_adc
        self.sw = sw_pin
        self.center_x = 0
        self.center_y = 0
        self.deadzone = 50  # mV or ADC counts

    def begin(self):
        """校准中心。"""
        # 采集中心位置
        sum_x = sum_y = 0
        for _ in range(100):
            sum_x += self.vrx.read()
            sum_y += self.vry.read()
        self.center_x = sum_x // 100
        self.center_y = sum_y // 100

    def read(self):
        """返回 (x, y) 归一化到 -100~+100。"""
        raw_x = self.vrx.read()
        raw_y = self.vry.read()
        x = raw_x - self.center_x
        y = raw_y - self.center_y
        # 死区
        if abs(x) < self.deadzone:
            x = 0
        if abs(y) < self.deadzone:
            y = 0
        # 归一化
        x_norm = x * 100 / (self.center_x)  # 假设量程对称
        y_norm = y * 100 / (self.center_y)
        return x_norm, y_norm

    def is_pressed(self):
        return self.sw.value() == 0
```

## 9. 数据格式与单位

- 模拟：电压（mV / ADC 计数）；
- 数字：归一化（-100~+100）；
- 角度：°（±30° 典型）；
- 状态：按下/未按下。


## 10. 校准与滤波

- **中心校准**：启动时采样 N 次取平均；
- **量程校准**：记录最大/最小值 → 归一化；
- **死区**：消除中心抖动（中心 ±5% 不响应）；
- **温度补偿**：电位器阻值受温度影响；
- 校准数据保存到 Flash。

### 11.3 死区 + 滞回

```python
def deadzone(value, threshold):
    if abs(value) < threshold:
        return 0
    return value

def hysteresis(value, last, threshold):
    if abs(value - last) < threshold:
        return last
    return value
```


## 11. 错误处理

- **抖动**：滤波；
- **中心漂移**：温度、机械磨损，重新校准；
- **按键抖动**：去抖；
- **最大电压不足**：供电电压低或电位器老化；
- **输出卡死**：机械磨损、电位器故障。


## 12. 低功耗

- 不需要持续采集，可定时采样；
- 模拟摇杆功耗极低（µA 级）；
- 霍尔摇杆 mA 级（测量时）。


## 13. 示例代码

```python
joy = Joystick(vrx_adc, vry_adc, sw_pin)
joy.begin()
while True:
    x, y = joy.read()
    print(f"X={x:.1f} Y={y:.1f}")
    if joy.is_pressed():
        print("按下")
    time.sleep_ms(50)
```


## 14. 调试方法

1. 用万用表测 VRx、VRy 电压（中心 2.5V@5V）；
2. 用示波器看波形；
3. 测 ADC 计数（0~4095）；
4. 测试最大行程电压；
5. 测试按键通断；
6. 测试滤波效果。


## 15. 常见问题

- **中心偏移**：电位器老化，重新校准；
- **抖动**：滤波不足，加硬件滤波电容；
- **方向反**：交换 X/Y；
- **按键不响应**：上拉未接、GPIO 配置错；
- **电位器失效**：机械磨损，需更换；
- **行程范围不足**：电位器阻值不匹配。


## 16. 参考资料

- 电位器寿命有限（10⁴~10⁶ 次）；
- 长时间使用后中心可能漂移；
- 高频震动环境建议用霍尔摇杆；
- 操作力反映手感，不同应用选不同操作力；
- 防尘防水需求选工业级。
