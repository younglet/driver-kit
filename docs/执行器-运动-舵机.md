# 舵机


## 1. 模块概览

舵机（Servo）是一种集成了直流电机 + 减速箱 + 控制电路 + 位置反馈的位置伺服器，通过 PWM 信号控制角度。常见于：
- 机器人关节；
- 遥控模型（车、船、飞机）；
- 舵机机械臂；
- 智能小车转向；
- 摄像头云台；
- DIY 玩具。

类型：
- **模拟舵机**：50 Hz PWM 控制；
- **数字舵机**：MCU 内置 PID，高速高精度；
- **塑料齿舵机**：低成本；
- **金属齿舵机**：高扭矩、长寿命；
- **总线舵机**：串行通信，多舵机级联（如 LewanSoul/Feetech SCS、AX、LX 系列）。


## 2. 工作原理与适用场景

内部结构：直流电机 + 减速齿轮组 + 位置电位器（反馈）+ 控制电路板。

输入 PWM：
- 50 Hz（20 ms 周期）；
- 脉宽 0.5~2.5 ms（部分舵机 1.0~2.0 ms）；
- 中位 1.5 ms = 90°（或舵机中心位置）；
- 0.5 ms = 0°，2.5 ms = 180°；
- 范围可调（如 270°、360°）。

控制电路将 PWM 脉宽与电位器位置对比，驱动电机至目标位置。

## 5. 硬件连接

- 三线接口：信号（橙/黄）、VCC（红）、GND（棕/黑）；
- 独立供电：舵机电流大，建议独立供电（5V/6V 电源，1A+）；
- 与 MCU 共地：必须共地；
- 信号电平：通常 3.3V~5V 兼容；
- PWM 频率：50 Hz（部分支持 200 Hz~300 Hz 高速舵机）；
- 续流二极管：电源加大容量电容（如 470µF）；
- 防止堵转烧毁：注意扭矩限制；
- 电流限制：保险丝或软件堵转保护。


## 6. 通信方式

### 6.1 PWM（模拟/数字舵机）

50 Hz，单脉宽控制角度。

### 6.2 总线舵机（串行）

- 一根数据线（半双工）；
- 多舵机级联（菊花链）；
- 协议：LewanSoul LSC、Feetech SCS、AX、LX、Dynamixel 等；
- 读写位置、速度、扭矩、温度、ID、波特率。


## 7. 初始化流程

### 7.1 PWM 舵机

1. 配置 PWM（50 Hz，0.5~2.5 ms 脉宽）；
2. 输出中位（1.5 ms）；
3. 控制信号角度变化。

### 7.2 总线舵机

1. 配置串口（9600/115200）；
2. 设置舵机 ID（出厂默认 1）；
3. 设置波特率（部分可调）；
4. 设置运动模式（位置/速度/扭矩）；
5. 写入位置指令；
6. 读取状态。

## 9. 数据格式与单位

### 9.1 PWM

脉宽 µs：500~2500 µs 对应 0~180°。

### 9.2 总线舵机

LewanSoul LSC 协议（飞特 Feetech 类似）：
- 帧：0x55 0x55 [长度] [命令] [参数...n] [校验和]
- 位置：0~1000（对应 0~240°）
- 速度：0~1000 + 方向
- 温度：°C
- 电压：×10 整数

Dynamixel 协议（更复杂）：
- 多指令读写；
- 间接寻址；
- 反馈：位置/速度/扭矩/温度/电压等。


## 10. 错误处理

- 抖动：信号不稳、电源纹波、脉宽毛刺；
- 不响应：信号频率错、脉宽超范围；
- 抖动在中位附近：死区调整；
- 总线舵机通信错误：检查 ID、波特率、接线、终端电阻；
- 堵转烧毁：扭矩保护、限位开关、堵转检测。


## 11. 低功耗

- 总线舵机支持关闭扭矩；
- PWM 舵机空闲时保持位置耗电；
- 降低工作电压节能（扭矩下降）。

## 13. 调试方法

1. 测试 0°/90°/180° 位置；
2. 检查扭矩（手测）；
3. 测试响应速度；
4. 检查信号波形（示波器）；
5. 总线舵机用串口监视器看指令。


## 14. 常见问题

- **抖动**：信号不稳，电源滤波；
- **不响应**：PWM 频率错、脉宽超范围；
- **无力**：电源电压不足、电流限制；
- **过载烧毁**：堵转时间过长；
- **总线舵机通信失败**：ID 冲突、波特率不匹配、接线错；
- **回中不准**：电位器老化或中心位置偏差；
- **间歇性丢位置**：电源电流不足。
