# 电源总览

电源链路管理是嵌入式系统的命脉。DriverKit 收录 **3 个电源模块**，覆盖电压电流监测和锂电充电。

## 模块速查

| 模块 | 说明 | 文档 |
|---|---|---|
| 电压监测 | 电阻分压 + ADC / 专用监测芯片 | [打开](./电源-监测-电压) |
| 电流监测 | 分流电阻 + 霍尔 / 专用芯片 | [打开](./电源-监测-电流) |
| 锂电充电器 | 线性 / 开关 / BMS / 无线 / PD | [打开](./电源-充电-锂电充电器) |

## 选型速查表

| 应用场景 | 推荐 |

## 共性知识

- **电池安全第一**：必须 BMS 保护板（过充/过放/短路）
- **电压监测**：分压电阻 + ADC，注意输入阻抗
- **电流监测**：
  - 高边测量保留共地
  - 4 线（开尔文）连接减少误差
- **充电四阶段**：涓流 → 恒流 → 恒压 → 终止
- **PD 诱骗**：CH224K / HUSB238 控制电压
