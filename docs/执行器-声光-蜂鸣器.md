# 蜂鸣器


## 1. 模块概览

蜂鸣器（Buzzer）是发声器件，用于报警、提示、音乐播放。常见于：
- 家电提示音；
- 门铃；
- 报警器；
- 倒计时器；
- 电子玩具；
- 仪器仪表；
- 智能家居；
- 简单音乐播放。

类型：
- **有源蜂鸣器**：内置振荡器，给电响（直流驱动）；
- **无源蜂鸣器**：需外部 PWM 驱动；
- **电磁式**：最常用；
- **压电式**：薄片振动，频率高；
- **压电扬声器**：连续播放语音/音乐。


## 2. 工作原理与适用场景

### 2.1 电磁式蜂鸣器

电流通过线圈产生磁场，驱动振膜振动发声。
- 有源：内部含振荡电路（典型 2~4 kHz）；
- 无源：外部提供 PWM 信号。

### 2.2 压电式蜂鸣器

压电材料（陶瓷）在外加电压时变形。给高频方波即可发声。

## 5. 硬件连接

### 5.1 有源蜂鸣器

- 直接 GPIO 驱动（高电平响）；
- 注意电流（5V 有源 ~30 mA，部分 GPIO 驱动不够，加三极管）；
- 模块版通常内置三极管，直接 IO 即可；
- 反并二极管（内部通常已有）。

### 5.2 无源蜂鸣器

- 需要外部 PWM 驱动；
- 占空比 50%；
- 频率对应音高；
- 同样注意驱动电流。

### 5.3 压电蜂鸣器

- 直接 IO 驱动（部分需电平转换）；
- 高频方波；
- 电流较小。

### 5.4 喇叭

- 需要功放（如 PAM8403）；
- 不能直接接 MCU；
- 8Ω 喇叭配 8Ω 输出功放。


## 6. 通信方式

- **GPIO**：有源蜂鸣器开关；
- **PWM**：无源蜂鸣器/压电音调控制；
- **I2S**：数字音频（高级播放）；
- **DAC + 功放**：模拟音频；
- **MP3 模块**：预存音频（如 DFPlayer Mini）。


## 7. 初始化流程

1. 配置 GPIO（输出模式）/ PWM 通道；
2. 设置初始为关闭；
3. 发声时给信号；
4. 关闭停止。


## 8. 驱动接口

```python
class Buzzer:
    """有源蜂鸣器。"""
    def __init__(self, pin, active_high=True):
        self.pin = pin
        self.active_high = active_high

    def begin(self):
        self.off()

    def on(self):
        if self.active_high:
            self.pin.high()
        else:
            self.pin.low()

    def off(self):
        if self.active_high:
            self.pin.low()
        else:
            self.pin.high()


class Tone:
    """无源蜂鸣器（PWM 驱动）。"""
    def __init__(self, pwm):
        self.pwm = pwm

    def begin(self):
        pass

    def tone(self, freq_hz, duration_ms=0):
        """播放指定频率。"""
        period_us = 1_000_000 // freq_hz
        self.pwm.freq(freq_hz)
        self.pwm.duty(50)  # 50% 占空比
        if duration_ms > 0:
            time.sleep_ms(duration_ms)
            self.pwm.duty(0)

    def no_tone(self):
        self.pwm.duty(0)


class Melody:
    """简单旋律播放。"""
    NOTES = {
        'C4': 262, 'D4': 294, 'E4': 330, 'F4': 349,
        'G4': 392, 'A4': 440, 'B4': 494,
        'C5': 523, 'D5': 587, 'E5': 659, 'F5': 698,
        'G5': 784, 'A5': 880, 'B5': 988,
        'REST': 0,
    }

    def __init__(self, tone):
        self.tone = tone

    def play_note(self, note, duration_ms):
        if note == 'REST':
            self.tone.no_tone()
        else:
            self.tone.tone(self.NOTES[note], duration_ms)
        time.sleep_ms(int(duration_ms * 0.3))  # 间隔
```

## 9. 数据格式与单位

- 有源：布尔；
- 无源：PWM 频率 Hz；
- MP3 模块：UART 指令。


## 10. 错误处理

- **不响**：电源未接、PWM 未输出、信号错；
- **声音小**：电压不足、驱动电流不够；
- **音质差**：频率不准、PWM 占空比非 50%；
- **杂音**：电源纹波、信号不稳。


## 11. 低功耗

- 不发声时关闭电源；
- 间歇发声节能；
- 音量调节（PWM 占空比影响音量）。

## 13. 调试方法

1. 测驱动电压/信号；
2. 听频率是否正确；
3. 测音量（手机分贝仪）；
4. 检查电流（是否足够驱动）；
5. 检查谐振频率是否匹配。


## 14. 常见问题

- **不响**：电源错、信号未输出；
- **声音小**：驱动电流不足；
- **音质差**：PWM 占空比非 50%、频率不准；
- **杂音**：电源纹波、信号抖动；
- **喇叭无声**：功放未接、接线错。
